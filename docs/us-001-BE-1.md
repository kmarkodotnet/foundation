## Story

> As a **bármely felhasználó**, I want to sign in with my Google account, so that I don't need to manage a separate password and can securely access grant data.

---

## Scope

Implements the Google OAuth 2.0 / OIDC backend flow: receiving the authorization code from the Angular client, exchanging it for Google tokens, provisioning or loading the `AppUser` record, issuing a signed internal JWT. Does NOT cover logout (us-002-BE-1) or RBAC enforcement (us-003-BE-1).

## API Contract

### Endpoint
```
POST /api/v1/auth/google-callback
```

### Authorization
- Required roles: None (anonymous)
- Guard: None

### Request Body
```json
{
  "authorizationCode": "4/0AX4XfWi...",
  "redirectUri": "https://palyazat.alapitvany.hu/auth/callback"
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `authorizationCode` | `string` | Yes | Not empty |
| `redirectUri` | `string` | Yes | Not empty |

### Responses

**200 OK:**
```json
{
  "accessToken": "eyJhbGci...",
  "expiresIn": 28800,
  "user": {
    "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "email": "user@gmail.com",
    "fullName": "Teszt Elek",
    "pictureUrl": "https://lh3.googleusercontent.com/...",
    "role": "Megtekinto",
    "lastLoginAt": "2025-01-01T08:00:00Z"
  }
}
```

**400** — Empty authorizationCode or redirectUri  
**401** — Google code exchange failed  
**403** — User account `IsActive == false`: `"A fiókod inaktív. Kérj segítséget az adminisztrátortól."`
