## Story

> As a **bármely felhasználó**, I want to sign in with my Google account, so that I don't need to manage a separate password and can securely access grant data.

---

## Scope

Login page with Google button, OIDC redirect initiation, OIDC callback component that exchanges the code for a JWT, JWT storage, `AuthService` singleton, `AuthGuard`. Does NOT cover logout (us-002-FE-1) or role-based guards (us-003-FE-1).

## UX / Behaviour Specification

| Scenario | Expected behaviour |
|---|---|
| Védett útvonalra látogat bejelentkezés nélkül | AuthGuard → `/login` redirect |
| Google gombra kattint | Browser → Google consent screen |
| Google visszairányít érvényes code-dal | OidcCallback → backend hívás → token tárolás → `/applications` |
| Backend 403 (inaktív) | `/login?error=inactive` → „A fiókod inaktív. Kérj segítséget az adminisztrátortól." |
| Backend 401 (rossz code) | `/login?error=auth_failed` → „Bejelentkezés sikertelen. Kérjük, próbálja újra." |
| Token lejár (8 óra) | Következő API hívás → 401 → AuthInterceptor → logout → `/login` |
