## Story

> As a **bejelentkezett felhasználó**, I want to log out, so that I can securely close my session.

---

## Scope

Implements the logout endpoint that invalidates the user's session token. Since the system uses stateless JWT, logout is handled by clearing the client-side token and optionally maintaining a server-side token denylist.

## API Contract

### Endpoint
```

```

### Authorization

- Guard: None

### Request

### Responses
- **204 No Content** — logout successful
