## Story

> As a **bejelentkezett felhasználó**, I want to log out, so that I can securely close my session.

---

## Scope

Logout button in the navbar profile menu, calls backend logout endpoint, clears local JWT, redirects to login page.

---

## Implementation Checklist

### Feature Module
- [ ] `src/app/layout/navbar/navbar.component.ts` + `.html` + `.scss`
  - Profil menü: felhasználó neve, e-mail, profilkép
  - „Kijelentkezés" menüpont (MatMenuItem)
  - Kattintásra `AuthService.logout()` hívás

### Service / API Client
- [ ] `AuthService.logout()` bővítése:
  - POST /api/v1/auth/logout hívás (best-effort, hiba esetén is folytatja)
  - `sessionStorage.removeItem('gm_token')`
  - `currentUser$.next(null)`
  - `router.navigate(['/login'])`
- [ ] `NavbarComponent` injektálja `AuthService`-t, `currentUser$`-t subscribe-olja a profilképhez

### Shared Components
- [ ] `MatMenuModule` — profil legördülő menü
- [ ] `MatIconModule` — felhasználó ikon

### Navigation
- [ ] Logout után: `router.navigate(['/login'])`
