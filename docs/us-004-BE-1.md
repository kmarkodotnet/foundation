## Story

> As a **bejelentkezett felhasználó**, I want to view my own profile data, so that I can verify the information stored about me in the system.

---

## Scope

GET /api/v1/auth/me endpoint returning the current authenticated user's profile. Read-only, no modification.

---

## API Contract

### Endpoint
```
GET /api/v1/auth/me
```

### Authorization
- Required roles: Any authenticated user

### Response (200 OK)
```json
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "email": "user@gmail.com",
  "fullName": "Teszt Elek",
  "pictureUrl": "https://lh3.googleusercontent.com/...",
  "role": "PalyazatiMunkatars",
  "lastLoginAt": "2025-01-01T08:00:00Z"
}
```

- **401** — not authenticated
