## Story

> As a **bejelentkezett felhasználó**, I want to view my own profile data, so that I can verify the information stored about me in the system.

---

## Scope

Profile page at `/profile` displaying read-only user information fetched from the API.

---

## Implementation Checklist

### Feature Module (`src/app/features/profile/`)
- [ ] `profile.module.ts` — lazy-loaded feature module
- [ ] `profile/profile.component.ts` + `.html` + `.scss`
  - Megjeleníti: profilkép, teljes név, e-mail, szerepkör, utolsó bejelentkezés
  - Csak olvasható (nincs szerkesztés)
- [ ] Route: `/profile` in `app-routing.module.ts` (canActivate: [AuthGuard])

### Service / API Client
- [ ] `src/app/features/profile/profile.service.ts`:
  - `getProfile(): Observable<UserProfileDto>` — GET /api/v1/auth/me
  - Eredmény cache-elve `shareReplay(1)` — ritkán változik session alatt
- [ ] `UserProfileDto` mindig a `src/app/core/auth/models/user-profile.model.ts`-ből importálandó — ne definiálj helyi másolatot; ha a modell még nincs ott, ott kell létrehozni

### State / Data Flow
- [ ] `ProfileComponent.ngOnInit()`: `profileService.getProfile()` hívás, eredmény megjelenítés
- [ ] Loading spinner amíg adat betöltődik
- [ ] Hiba esetén toast: „Nem sikerült betölteni a profil adatokat."

### Navigation
- [ ] Navbar profil ikonra kattintva navigál `/profile`-ra
