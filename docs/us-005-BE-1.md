## Story

> As a **bejelentkezett felhasználó**, I want to configure which notifications I receive by email, so that only relevant notifications reach me.

---

## Scope

PUT endpoint to update the current user's notification preferences (email toggles). Preferences are stored in `AppUser.NotificationPreferences`.

---

## API Contract

### Endpoint
```
PUT /api/v1/auth/me/notification-preferences
```

### Request Body
```json
{
  "submissionDeadlineEmail": true,
  "spendingDeadlineEmail": false,
  "approvalRequestEmail": true
}
```

### Response (200 OK)
```json
{
  "submissionDeadlineEmail": true,
  "spendingDeadlineEmail": false,
  "approvalRequestEmail": true
}
```

- **401** — not authenticated
