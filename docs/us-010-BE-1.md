## Story

> As a **pályázati munkatárs**, I want to record a new grant call, so that we can track the grant opportunity and start the submission process.

---

## Scope

Creates a new Application entity in DRAFT status with auto-generated 9 WorkflowSteps. The first step (Call) immediately becomes ACTIVE. Includes Granter autocomplete support.

## API Contract

### Endpoint
```
POST /api/v1/applications
```

### Request Body
```json
{
  "title": "Oktatási pályázat 2025",
  "granterId": "uuid",
  "submissionDeadline": "2025-06-30T23:59:59Z",
  "identifier": "PAL-2025-001",
  "description": "...",
  "applicationTypeId": "uuid",
  "minAmount": 500000,
  "maxAmount": 2000000,
  "spendingDeadline": "2025-12-31",
  "otherMetadata": "..."
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `title` | `string` | Yes | Not empty, max 500 |
| `granterId` | `Guid` | Yes | Must exist |
| `submissionDeadline` | `DateTimeOffset` | Yes | Must be in future |
| `identifier` | `string?` | No | Max 100 |
| `description` | `string?` | No | Max 5000 |

### Response (201 Created)
```json
{
  "id": "uuid",
  "title": "Oktatási pályázat 2025",
  "status": "Draft",
  "granterId": "uuid",
  "granterName": "Emberi Erőforrások Minisztériuma",
  "submissionDeadline": "2025-06-30T23:59:59Z",
  "workflowSteps": [
    { "stepType": "Call", "status": "Active", "order": 1 },
    { "stepType": "Submission", "status": "Pending", "order": 2 }
  ],
  "createdAt": "2025-01-01T10:00:00Z"
}
```

- **400** — validation error
- **401** — not authenticated
- **403** — insufficient role
- **404** — GranterId not found
- **422** — SubmissionDeadline in the past

## Notes
- A `WorkflowService.CreateWorkflowSteps()` domain service-ként implementálandó, nem a handlerben inline.
- IsSkippable: ContractGranter=true, VendorContracts=true, Proof=true; a többi false.
- `StepData` (TEXT, nullable): step-specifikus mezőket JSON-ként tárol (pl. Submission: SubmittedAt, SubmissionMethodId, ExternalIdentifier, Notes). Ily módon a WorkflowStep tábla nem bővül lépésenként külön oszlopokkal.
- `Application.IsLocked()` = `Status is ClosedWon or ClosedLost or Archived`; az `IApplicationCommand` pipeline behaviour ezt hívja meg a guard checkhez.
