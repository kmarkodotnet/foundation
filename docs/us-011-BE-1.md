## Story

## API Contract

### Endpoint
```
PUT /api/v1/applications/{id}
```

### Request Body
```json
{
  "title": "Frissített cím",
  "submissionDeadline": "2025-07-15T23:59:59Z"
}
```
(Partial update — only provided fields are changed)

### Responses
- **200 OK** — updated ApplicationDto
- **400** — validation error
- **403** — LOCKED application + non-Admin
- **404** — application not found
- **409** — optimistic concurrency conflict (RowVersion mismatch)
