## Story

> As a **bármely jogosult felhasználó**, I want to view the details of a grant call, so that I can learn about the grant conditions.

---

## Scope

GET endpoint returning full application details including all workflow steps and their current status. Available to all authenticated users.

---

## API Contract

### Endpoint
```
GET /api/v1/applications/{id}
GET /api/v1/applications?page=1&pageSize=20&status=Draft
```

### Response (200 OK) — GET /{id}
```json
{
  "id": "uuid",
  "title": "Oktatási pályázat 2025",
  "status": "Draft",
  "granterName": "...",
  "submissionDeadline": "2025-06-30T23:59:59Z",
  "awardedAmount": null,
  "workflowSteps": [
    { "stepType": "Call", "status": "Active", "order": 1, "isSkippable": false },
    { "stepType": "Submission", "status": "Pending", "order": 2, "isSkippable": false }
  ],
  "createdAt": "2025-01-01T10:00:00Z",
  "updatedAt": "2025-01-01T10:00:00Z"
}
```

- **401** — not authenticated
- **404** — not found
