## Story

> As a **bármely jogosult felhasználó**, I want to view the details of a grant call.

---

## Scope

Read-only application detail view with workflow stepper showing all 9 steps and their statuses. The call step content is displayed in the first step panel.

---

## Implementation Checklist

### Feature Module
- [ ] `detail/application-detail.component.ts` (established in us-010-FE-1) — bővítés: step-call tartalom megjelenítése
- [ ] `detail/workflow/step-call/step-call.component.ts` + `.html` — felhívás adatainak megjelenítése (read-only mód)
- [ ] `shared/components/status-badge/status-badge.component.ts` — lépés és pályázat státusz badge (Pending=szürke, Active=kék, Completed=zöld, Skipped=narancs, Locked=piros)

### State / Data Flow
- [ ] `ApplicationDetailComponent` — állapotgép: aktív lépés kiszámítása, navigáció a lépések között
- [ ] WorkflowStepper: `MatStepper` vagy custom timeline komponens a 9 lépéshez
- [ ] `ActivatedRoute.queryParams` figyelése: `?step=<stepIndex>` query param esetén automatikusan a megadott lépésre navigál (pl. értesítési linkekből nyílik meg)

### Shared Components
- [ ] `<app-status-badge [status]="application.status">` badge a fő pályázat állapotnál
