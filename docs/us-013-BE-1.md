## Story

> As a **adminisztrátor**, I want to archive a grant application, so that inactive applications don't clutter the active list.

---

## Scope

DELETE (soft) endpoint to archive an application. Sets IsArchived=true. Admin only.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Applications/Commands/ArchiveApplication/ArchiveApplicationCommand.cs` — Guid ApplicationId
- [ ] Handler: `Application/Applications/Commands/ArchiveApplication/ArchiveApplicationCommandHandler.cs` — loads Application, sets IsArchived=true, saves
- [ ] No validator needed (Id only)
- [ ] Implements `IApplicationCommand`

### API Layer
- [ ] Action in `ApplicationsController`:
  - `DELETE /api/v1/applications/{id}` (soft delete = archive)
  - `[Authorize]`, `[RequireRole(UserRole.Admin)]`
  - Returns `204 No Content`
- [ ] `[ProducesResponseType(204)]`, 401, 403, 404

### Authorization
- [ ] Admin only

---

## API Contract

```
DELETE /api/v1/applications/{id}
```

- **204** — archived successfully
- **401** — not authenticated
- **403** — non-Admin
- **404** — not found
