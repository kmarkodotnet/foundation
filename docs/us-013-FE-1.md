## Story

> As a **adminisztrátor**, I want to archive a grant application so that inactive ones don't clutter the list.

---

## Scope

Archive button on the application detail/list, confirm dialog, calls DELETE API, removes from active list.

---

## Implementation Checklist

### Feature Module
- [ ] „Archiválás" gomb hozzáadása `ApplicationDetailComponent`-hez és a lista sorhoz
  - `*hasRole="['Admin']"` direktíva
  - Kattintásra `ConfirmDialogComponent` nyílik

### Service
- [ ] `ApplicationsService.archiveApplication(id: string): Observable<void>` — DELETE /api/v1/applications/:id

### Shared Components
- [ ] `src/app/shared/components/confirm-dialog/confirm-dialog.component.ts` — **ez a task hozza létre** (minden más ticket erre hivatkozik):
  ```typescript
  @Component({ selector: 'app-confirm-dialog' })
  export class ConfirmDialogComponent {
    @Input() title: string = 'Megerősítés';
    @Input() message: string = '';
    @Input() confirmText: string = 'Igen';
    @Input() cancelText: string = 'Mégsem';
  }
  ```
  - Export from `SharedModule`
  - Használat: `MatDialog.open(ConfirmDialogComponent, { data: { title, message, confirmText } })`
  - Visszatér: `true` ha megerősítve, `false` / `undefined` ha elutasítva
- [ ] Archiválás szöveg: „Biztosan archiválni szeretnéd ezt a pályázatot? Az archivált pályázatok az alapértelmezett listából eltűnnek."

### State / Data Flow
- [ ] Sikeres archiválás után: toast „Pályázat archiválva.", navigáció `/applications`-re
