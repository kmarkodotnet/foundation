## Story

> As a **pályázati munkatárs**, I want to record the submission details, so that the submission fact and date are documented.

---

## Scope

Updates the Submission WorkflowStep with submission data, advances application status to SUBMITTED, and sends an approval notification to the president.

## API Contract

### Endpoint
```
PUT /api/v1/applications/{id}/workflow/submission
```

### Request Body
```json
{
  "submittedAt": "2025-06-29T14:30:00Z",
  "submissionMethodId": "uuid",
  "externalIdentifier": "REF-2025-001",
  "notes": "Postán beadva"
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `submittedAt` | `DateTimeOffset` | Yes | Not empty, not in future |

### Response (200 OK)
```json
{
  "stepType": "Submission",
  "status": "Active",
  "submittedAt": "2025-06-29T14:30:00Z",
  "submissionMethodName": "Postai úton"
}
```

- **400** — missing submittedAt
- **403** — LOCKED or wrong role
- **404** — application not found
- **422** — step not in ACTIVE state

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `SubmittedAt` | `NotEmpty()` | „A beadás időpontja kötelező." |
| `SubmittedAt` | `LessThanOrEqualTo(DateTimeOffset.UtcNow)` | „A beadás időpontja nem lehet jövőbeli." |
