## Story

> As a **elnök**, I want to approve the submission, so that it is confirmed at the presidential board level.

---

## Scope

Approve or reject the Submission workflow step. Approval completes the step; rejection keeps it ACTIVE with a note.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Workflow/Commands/ApproveStep/ApproveStepCommand.cs` — Guid ApplicationId, WorkflowStepType StepType, bool IsApproved, string? RejectionNote
- [ ] Handler: `Application/Workflow/Commands/ApproveStep/ApproveStepCommandHandler.cs`:
  - Loads WorkflowStep
  - If IsApproved: sets Status=Completed, ApprovedAt, ApprovedByUserId
  - If !IsApproved: keeps Status=Active, stores RejectionNote
- [ ] Validator: ApproveStepCommandValidator — if !IsApproved, RejectionNote required
- [ ] Implements `IApplicationCommand`

### API Layer
- [ ] Action in `WorkflowController`:
  - `POST /api/v1/applications/{id}/workflow/{stepType}/approve`
  - `[Authorize]`, `[RequireRole(UserRole.Admin, UserRole.Elnok)]`
  - Returns `200 OK` with `WorkflowStepDetailDto`

### Authorization
- [ ] Elnok, Admin only

---

## API Contract

```
POST /api/v1/applications/{id}/workflow/submission/approve
```

### Request Body
```json
{ "isApproved": true, "rejectionNote": null }
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `isApproved` | `bool` | Yes | — |
| `rejectionNote` | `string?` | Conditional | Required if isApproved=false |

- **200** — WorkflowStepDetailDto
- **403** — not Elnok/Admin
- **422** — step not in state that allows approval

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `RejectionNote` | `NotEmpty() when !IsApproved` | „Visszautasítás esetén meg kell adni az okát." |
