## Story

> As a **pályázati munkatárs**, I want to record that we won the grant and provide the awarded amount, so that winner application data is complete.

---

## Scope

Sets application status to WON, records awarded amount and result details, activates steps 4–9.

## API Contract

```
PUT /api/v1/applications/{id}/workflow/result
```

### Request Body
```json
{
  "isWon": true,
  "awardedAmount": 1500000,
  "resultDate": "2025-08-01",
  "resultIdentifier": "ERED-2025-001"
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `isWon` | `bool` | Yes | — |
| `awardedAmount` | `decimal?` | If isWon=true | >0 |
| `resultDate` | `DateOnly?` | If isWon=true | Not in future |

- **200** — ApplicationDto with Won status and activated steps
- **422** — invalid state transition

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `AwardedAmount` | `GreaterThan(0) when IsWon` | „Az elnyert összegnek pozitívnak kell lennie." |
| `ResultDate` | `NotEmpty() when IsWon` | „Az eredmény dátuma kötelező." |
