## Story

> As a **pályázati munkatárs**, I want to record that we lost the grant so that the application lifecycle closes and it becomes archivable.

---

## Scope

Uses the same `RecordResultCommand` as US-030 (isWon=false branch). Sets application status to LOST, marks steps 4–9 as NOT_APPLICABLE.

---

## API Contract

```
PUT /api/v1/applications/{id}/workflow/result
```

### Request Body (Lost path)
```json
{
  "isWon": false,
  "awardedAmount": null,
  "resultDate": "2025-08-01",
  "resultIdentifier": null
}
```

```
PUT /api/v1/applications/{id}/workflow/close-lost
```
No request body. Returns `ApplicationDto` with status ClosedLost and all steps LOCKED.

| Response | Description |
|---|---|
| **200** | ApplicationDto with LOST status, steps 4–9 NOT_APPLICABLE |
| **422** | Invalid state transition |
| **403** | Insufficient role for close-lost |
