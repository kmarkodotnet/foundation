## Story

> As a **pályázati munkatárs**, I want to record that we lost the grant so that the application lifecycle closes.

---

## Scope

Extends the shared `StepResultComponent` (built in US-030) with the Lost path: confirmation dialog, no amount fields, and a "Close Application" button for Admin/Elnök.

---

## Implementation Checklist

### Feature Module
- [ ] `StepResultComponent` (existing from US-030) — Lost path:
  - „Nem nyert" radio selection → megerősítő dialog
  - Hide amount/date fields
  - „Lezárás CLOSED_LOST-ra" gomb (Admin/Elnök, `*hasRole`)
  - ConfirmDialogComponent reuse

### Service
- [ ] `WorkflowService.closeLost(appId: string): Observable<ApplicationDto>` — PUT /api/v1/applications/:id/workflow/close-lost

### Reactive Form
- [ ] `isWon = false` path: `awardedAmount` and `resultDate` hidden + validators reset
- [ ] `resultDate`: optional (no required validator when isWon=false)

### Authorization
- [ ] „Lezárás" gomb: `*hasRole="['Admin', 'Elnok']"`

### State / Data Flow
- [ ] Sikeres mentés (nem nyert): Toast „Nem nyert eredmény rögzítve. A pályázat LOST státuszba került."
- [ ] Lépések [4]–[9] megjelennek „Nem alkalmazható" felirattal, szürkítve
- [ ] Sikeres lezárás: Toast „Pályázat CLOSED_LOST állapotba került.", navigáció a lista oldalra
