## Story

> As an **adminisztrátor**, I want to correct a mistakenly recorded result so that the application reflects the actual decision.

---

## Scope

Admin-only correction endpoint that re-runs the result recording logic, handling Won→Lost and Lost→Won transitions including cascading step status changes.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Workflow/Commands/CorrectResult/CorrectResultCommand.cs` — Guid ApplicationId, bool IsWon, decimal? AwardedAmount, DateOnly? ResultDate, string? ResultIdentifier
- [ ] Handler: `Application/Workflow/Commands/CorrectResult/CorrectResultCommandHandler.cs`:
  - Guards: Application.Status must not be ClosedWon or ClosedLost (unless actor is Admin)
  - If Won→Lost: calls `WorkflowService.HandleLostResult()` (steps 4–9 → NOT_APPLICABLE)
  - If Lost→Won: calls `WorkflowService.HandleWonResult()` (steps 4–9 → Active), sets AwardedAmount
  - Resets Result WorkflowStep data
- [ ] Validator: `CorrectResultCommandValidator` — same rules as RecordResultCommandValidator; Admin only
- [ ] Implements `IApplicationCommand`

### API Layer
- [ ] Action in `WorkflowController`:
  - `PUT /api/v1/applications/{id}/workflow/result/correct`
  - `[Authorize]`, `[RequireRole(UserRole.Admin)]`
  - Returns `200 OK` with `ApplicationDto`

### Authorization
- [ ] Admin only

---

## API Contract

```
PUT /api/v1/applications/{id}/workflow/result/correct
```

### Request Body
```json
{
  "isWon": true,
  "awardedAmount": 1500000,
  "resultDate": "2025-08-01",
  "resultIdentifier": "ERED-2025-001"
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `isWon` | `bool` | Yes | — |
| `awardedAmount` | `decimal?` | If isWon=true | >0 |
| `resultDate` | `DateOnly?` | If isWon=true | Not in future |

| Response | Description |
|---|---|
| **200** | ApplicationDto with corrected status |
| **403** | Not Admin |
| **422** | Application is CLOSED — cannot correct |

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `AwardedAmount` | `GreaterThan(0) when IsWon` | „Az elnyert összegnek pozitívnak kell lennie." |
| `ResultDate` | `NotEmpty() when IsWon` | „Az eredmény dátuma kötelező." |
