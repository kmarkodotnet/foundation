## Story

> As a **pályázati munkatárs**, I want to record the notification and contract data received from the granter so that the support agreement is documented.

---

## Scope

Updates the ContractGranter workflow step with contract and notification details. Step is only accessible when application is WON. Supports manual completion and skip-with-reason.

---

## API Contract

```
PUT /api/v1/applications/{id}/workflow/contract-granter
```

### Request Body
```json
{
  "contractIdentifier": "SZERZ-2025-001",
  "contractDate": "2025-09-01",
  "notificationReceived": true,
  "notificationDate": "2025-08-20",
  "complete": false
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `contractIdentifier` | `string?` | No | — |
| `contractDate` | `DateOnly?` | No | — |
| `notificationReceived` | `bool` | Yes | — |
| `notificationDate` | `DateOnly?` | No | — |
| `complete` | `bool` | Yes | — |

| Response | Description |
|---|---|
| **200** | WorkflowStepDetailDto |
| **422** | Application not in WON status |
