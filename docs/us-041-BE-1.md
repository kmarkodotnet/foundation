## Story

> As a **pályázati munkatárs**, I want to skip a skippable workflow step with an optional reason so that the process continues without being blocked.

---

## Scope

Generic skip command applicable to all skippable steps (ContractGranter, VendorContracts, Proof). Sets step status to SKIPPED and stores optional skip reason. Restorable to ACTIVE by Admin/Elnök.

---

## API Contract

```
POST /api/v1/applications/{id}/workflow/{stepType}/skip
```

### Request Body
```json
{
  "skipReason": "A pályáztató nem köt formális szerződést."
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `skipReason` | `string?` | No | Max 500 chars |

```
POST /api/v1/applications/{id}/workflow/{stepType}/restore
```
No request body. Returns `WorkflowStepDetailDto` with Status=Active.

| Response | Description |
|---|---|
| **200** | WorkflowStepDetailDto |
| **422** | Step is not skippable or not in skippable state |
