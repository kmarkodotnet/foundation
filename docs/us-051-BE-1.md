## Story

> As a **pályázati munkatárs**, I want to edit or delete budget items so that the plan can be updated as circumstances change.

---

## Scope

Update and soft-delete individual BudgetItems. Deletion blocked if linked invoices or vendor contracts exist.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/BudgetPlan/Commands/UpdateBudgetItem/UpdateBudgetItemCommand.cs` — Guid ApplicationId, Guid ItemId, string Name, BudgetItemType Type, decimal PlannedAmount, string? Description; implements `IApplicationCommand`
- [ ] Handler: loads BudgetItem by id, updates fields
- [ ] Command: `Application/BudgetPlan/Commands/DeleteBudgetItem/DeleteBudgetItemCommand.cs` — Guid ApplicationId, Guid ItemId; implements `IApplicationCommand`
- [ ] Handler: `DeleteBudgetItemCommandHandler`:
  - Checks for linked Invoices (Invoice.BudgetItemId == ItemId)
  - Checks for linked VendorContracts (VendorContract.BudgetItemId == ItemId)
  - If linked: throws DomainException → 422
  - Else: soft delete (IsDeleted=true)
- [ ] Validators: UpdateBudgetItemCommandValidator (same rules as create)

### API Layer
- [ ] Actions in `BudgetPlanController`:
  - `PUT /api/v1/applications/{id}/budget-plan/items/{itemId}` — `[RequireRole(Admin, PalyazatiMunkatars)]`
  - `DELETE /api/v1/applications/{id}/budget-plan/items/{itemId}` — `[RequireRole(Admin, PalyazatiMunkatars)]`
  - Update returns `200 OK` with `BudgetItemDto`
  - Delete returns `204 No Content`

### Authorization
- [ ] Admin, PalyazatiMunkatars

---

## API Contract

```
PUT /api/v1/applications/{id}/budget-plan/items/{itemId}
```

### Request Body
```json
{
  "name": "Nyári tábor (módosított)",
  "type": "Event",
  "plannedAmount": 600000,
  "description": "Frissített leírás"
}
```

```
DELETE /api/v1/applications/{id}/budget-plan/items/{itemId}
```

| Response | Description |
|---|---|
| **200** | BudgetItemDto (update) |
| **204** | No content (delete) |
| **422** | Item has linked invoices or contracts (delete blocked) |
