## Story

> As an **elnök**, I want to approve the budget plan so that spending is confirmed at the presidential level before it begins.

---

## Scope

Reuses the generic `ApproveStepCommand` (from US-021) for the BudgetPlan step type. Sends an approval request notification to Elnök; approval completes the step.

---

## API Contract

```
POST /api/v1/applications/{id}/workflow/budget-plan/request-approval
```
No body. Returns 204.

```
POST /api/v1/applications/{id}/workflow/budget-plan/approve
```

### Request Body
```json
{ "isApproved": true, "rejectionNote": null }
```

| Response | Description |
|---|---|
| **204** | Approval request sent |
| **200** | WorkflowStepDetailDto (approve/reject) |
| **403** | Insufficient role |
| **422** | No budget items exist (cannot request approval) |

## FluentValidation Rules
