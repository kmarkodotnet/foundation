## Story

> As a **pályázati munkatárs**, I want to record a vendor contract so that the service engagement for the grant is documented.

---

## Scope

Creates a VendorContract record linked to an Application. References a Vendor entity and optionally a BudgetItem. Multiple contracts per application. Step is skippable.

## API Contract

```
POST /api/v1/applications/{id}/vendor-contracts
```

### Request Body
```json
{
  "vendorId": "uuid",
  "contractDate": "2025-10-01",
  "amount": 250000,
  "identifier": "ALVSZ-2025-001",
  "budgetItemId": "uuid",
  "notes": null
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `vendorId` | `Guid` | Yes | Must exist |
| `contractDate` | `DateOnly` | Yes | — |
| `amount` | `decimal` | Yes | >0 |
| `identifier` | `string?` | No | — |
| `budgetItemId` | `Guid?` | No | Must exist if provided |

| Response | Description |
|---|---|
| **201** | VendorContractDto |
| **422** | Application not WON |

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `VendorId` | `NotEmpty()` | „A szerződő cég megadása kötelező." |
| `ContractDate` | `NotEmpty()` | „A szerződéskötés időpontja kötelező." |
| `Amount` | `GreaterThan(0)` | „Az összegnek pozitívnak kell lennie." |
