## Story

> As a **pályázati munkatárs**, I want to delete a mistakenly recorded vendor contract so that inaccurate data doesn't burden the grant records.

---

## Scope

Soft-deletes a VendorContract. Blocked if any invoices are linked to the contract.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/VendorContracts/Commands/DeleteVendorContract/DeleteVendorContractCommand.cs` — Guid ContractId
- [ ] Handler:
  - Checks for linked Invoices (Invoice.VendorContractId == ContractId)
  - If linked: throws DomainException → 422 with count
  - Else: soft delete (IsDeleted=true)
- [ ] Implements `IApplicationCommand`

### API Layer
- [ ] `DELETE /api/v1/applications/{id}/vendor-contracts/{contractId}`
  - `[RequireRole(UserRole.Admin, UserRole.PalyazatiMunkatars)]`
  - Returns `204 No Content`

### Authorization
- [ ] Admin, PalyazatiMunkatars

---

## API Contract

```
DELETE /api/v1/applications/{id}/vendor-contracts/{contractId}
```

| Response | Description |
|---|---|
| **204** | Deleted |
| **422** | Has linked invoices: „A szerződés nem törölhető, mert X db számla kapcsolódik hozzá." |
