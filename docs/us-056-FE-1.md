## Story

> As a **pályázati munkatárs**, I want to delete a mistakenly recorded vendor contract so that inaccurate data is removed.

---

## Scope

Delete action on each VendorContract row in `StepVendorContractsComponent`. Confirmation dialog; 422 error handling for linked invoices.

---

## Implementation Checklist

### Feature Module
- [ ] `StepVendorContractsComponent` — törlés ikon soronként:
  - ConfirmDialogComponent: összeg és azonosító megjelenítve
  - 422 hiba kezelés: error toast „A szerződés nem törölhető, mert X db számla kapcsolódik hozzá."
  - Sikeres törlés után: toast „Szerződés törölve.", elem eltávolítása a lokális tömbből (nem API újrahívás), összesítő frissítés

### Service
- [ ] `VendorContractService.deleteContract(appId: string, contractId: string): Observable<void>` — DELETE /api/v1/applications/:id/vendor-contracts/:contractId
