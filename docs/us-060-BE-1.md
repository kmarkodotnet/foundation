## Story

> As a **pénzügyes**, I want to record a new invoice so that financial performance is documented and trackable.

---

## Scope

Creates an Invoice record linked to an Application. Optional links to VendorContract and BudgetItem. Conditional validation: if IsPaid=true, PaymentDate is required.

## API Contract

```
POST /api/v1/applications/{id}/invoices
```

### Request Body
```json
{
  "supplierName": "Acme Kft.",
  "invoiceNumber": "SZ-2025-001",
  "issueDate": "2025-10-15",
  "amount": 125000,
  "isPaid": true,
  "paymentDate": "2025-10-20",
  "vendorContractId": null,
  "budgetItemId": "uuid",
  "notes": null
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `supplierName` | `string` | Yes | NotEmpty |
| `invoiceNumber` | `string` | Yes | NotEmpty |
| `issueDate` | `DateOnly` | Yes | — |
| `amount` | `decimal` | Yes | >0 |
| `isPaid` | `bool` | Yes | — |
| `paymentDate` | `DateOnly?` | If isPaid=true | ≥ issueDate |

| Response | Description |
|---|---|
| **201** | InvoiceDto |
| **422** | Application not WON |

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `SupplierName` | `NotEmpty()` | „A szállító neve kötelező." |
| `InvoiceNumber` | `NotEmpty()` | „A számla sorszáma kötelező." |
| `Amount` | `GreaterThan(0)` | „Az összegnek pozitívnak kell lennie." |
| `PaymentDate` | `NotEmpty() when IsPaid` | „Fizetés dátuma kötelező, ha a számla fizetve van." |
