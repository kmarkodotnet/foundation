## Story

> As a **pénzügyes**, I want to update an invoice's payment status so that the financial records stay current.

---

## Scope

Marks an invoice as paid with a payment date. Quick-update endpoint for the list view action.

---

---

## API Contract

```
PATCH /api/v1/applications/{id}/invoices/{invoiceId}/mark-paid
```

### Request Body
```json
{ "paymentDate": "2025-10-22" }
```

| Response | Description |
|---|---|
| **200** | InvoiceDto with IsPaid=true |
| **422** | Invoice already paid |
