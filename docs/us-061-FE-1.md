## Story

> As a **pénzügyes**, I want to quickly update an invoice's payment status so that records stay current.

---

## Scope

Quick-action "Mark as Paid" button in the invoice list row. Opens a mini-dialog for payment date entry. Visual status update after save.

---

## Implementation Checklist

### Feature Module
- [ ] `StepInvoicesComponent` — soronként:
  - „Megjelölés fizetettnek" gyorsgomb (csak ha IsPaid=false, `*hasRole="['Admin', 'Penzugyes']"`)
  - Zöld ikon/badge ha IsPaid=true

- [ ] `MarkPaidDialogComponent` (MatDialog, mini):
  - Fizetés dátuma (DatePicker, required)
  - „Mentés" + „Mégse" gombok

### Service
- [ ] `InvoiceService.markPaid(appId: string, invoiceId: string, paymentDate: string): Observable<InvoiceDto>` — PATCH /api/v1/applications/:id/invoices/:invoiceId/mark-paid
- [ ] `InvoiceService.updateInvoice(appId: string, invoiceId: string, data: UpdateInvoiceRequest): Observable<InvoiceDto>` — PUT /api/v1/applications/:id/invoices/:invoiceId (teljes rekord módosítása)

### Authorization
- [ ] „Megjelölés fizetettnek": `*hasRole="['Admin', 'Penzugyes']"`

### State / Data Flow
- [ ] Dialog mentés után: InvoiceDto frissítése helyi listában, összesítő újraszámítás
- [ ] Toast „Számla fizetettre jelölve."

## API Consumed

| Method | URL | Body | Response |
|---|---|---|---|
| `PATCH` | `/api/v1/applications/:id/invoices/:invoiceId/mark-paid` | `{ paymentDate }` | `InvoiceDto` |
| `PUT` | `/api/v1/applications/:id/invoices/:invoiceId` | `UpdateInvoiceRequest` | `InvoiceDto` |
