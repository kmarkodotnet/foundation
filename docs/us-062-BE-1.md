## Story

> As a **pénzügyes**, I want to delete a mistakenly recorded invoice so that errors don't distort financial summaries.

---

## Scope

Soft-deletes an Invoice. Blocked when application is LOCKED. Audit trail preserved.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Invoices/Commands/DeleteInvoice/DeleteInvoiceCommand.cs` — Guid InvoiceId
- [ ] Handler:
  - Checks application is not LOCKED
  - Soft delete (IsDeleted=true)
- [ ] Implements `IApplicationCommand`

### API Layer
- [ ] `DELETE /api/v1/applications/{id}/invoices/{invoiceId}`
  - `[RequireRole(UserRole.Admin, UserRole.Penzugyes)]`
  - Returns `204 No Content`

### Authorization
- [ ] Admin, Penzugyes

---

## API Contract

```
DELETE /api/v1/applications/{id}/invoices/{invoiceId}
```

| Response | Description |
|---|---|
| **204** | Deleted |
| **422** | Application is LOCKED |
