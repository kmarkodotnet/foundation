## Story

> As a **pénzügyes**, I want to delete a mistakenly recorded invoice so that errors don't affect financial summaries.

---

## Scope

Delete action on each Invoice row. Confirmation dialog shows amount and invoice number. Summary panel updates after deletion.

---

## Implementation Checklist

### Feature Module
- [ ] `StepInvoicesComponent` — törlés ikon soronként:
  - ConfirmDialogComponent: összeg és sorszám megjelenítve
  - 422 hiba: error toast „A pályázat zárolt állapotban van, törlés nem lehetséges."
  - Sikeres törlés: lista frissítése, összesítő frissítés

### Service
- [ ] `InvoiceService.deleteInvoice(appId: string, invoiceId: string): Observable<void>` — DELETE /api/v1/applications/:id/invoices/:invoiceId

### Authorization
