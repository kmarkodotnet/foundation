## Story

> As a **pénzügyes or elnök**, I want to see the full financial summary at a glance.

---

## Scope

Financial summary panel at the top of the Invoices step. Reactive to invoice list changes. Filterable/sortable invoice list below.

---

## Implementation Checklist

### Feature Module
- [ ] `StepInvoicesComponent` — összesítő panel:
  - MatCard: Elnyert összeg, Tervezett összeg, Rögzített számlák, Fizetve, Fizetetlen, Egyenleg
  - Egyenleg piros ha számlák összege > elnyert összeg
  - Panel automatikusan frissül műveletek után

- [ ] Szűrő és rendezés toolbar:
  - Fizetve/Nem fizetve/Összes MatButtonToggle szűrő
  - Rendezés: dátum ↑↓, összeg ↑↓ (MatSort)

### Service
- [ ] `InvoiceService.getInvoices(appId: string, filters?: InvoiceFilters): Observable<InvoiceListDto>` — already defined in US-060, extend with filter params
- [ ] `InvoiceListDto` TypeScript interfész:
  ```ts
  interface InvoiceListDto {
    summary: InvoiceSummaryDto;
    items: InvoiceDto[];
  }
  interface InvoiceSummaryDto {
    awardedAmount: number;
    totalPlanned: number;
    totalInvoiced: number;
    totalPaid: number;
    totalUnpaid: number;
    balance: number; // awardedAmount - totalInvoiced
  }
  ```

### State / Data Flow
- [ ] `InvoiceSummaryDto` megjelenítése reaktívan
- [ ] Szűrő változásakor: API hívás újrafuttatása
