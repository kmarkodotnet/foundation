## Story

> As a **pályázati munkatárs**, I want to record proof that an event occurred or a physical item was delivered, with photos, so that performance is documented toward the granter.

---

## Scope

Creates a ProofRecord linked to an Application. Requires at least one photo file. Multiple proof records per application. Step is skippable.

## API Contract

```
POST /api/v1/applications/{id}/proof-records
Content-Type: multipart/form-data
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `type` | `string` | Yes | Event or Asset |
| `occurrenceDate` | `DateOnly` | Yes | — |
| `notes` | `string?` | No | — |
| `photos` | `IFormFile[]` | Yes | Min 1, accepted: JPG/PNG/TIFF, max 50 MB each |

| Response | Description |
|---|---|
| **201** | ProofRecordDto |
| **422** | No photos provided or application not WON |

---
