## Story

> As an **elnök or megtekintő**, I want to view and download proof photos so that I can verify the performance.

---

## Scope

Secure photo download endpoints. Individual file download and ZIP bundle for all photos in a proof record. Files served through API (no direct URL access).

---

## Implementation Checklist

### Application Layer
- [ ] Query: `GetProofPhotoQuery` — Guid ProofPhotoId → returns file stream + content type
- [ ] Query: `DownloadAllProofPhotosQuery` — Guid ProofRecordId → returns ZIP stream of all photos
- [ ] Uses `IFileStorageService.ReadAsync(path): Task<Stream>`

### API Layer
- [ ] `GET /api/v1/applications/{id}/proof-records/{recordId}/photos/{photoId}` — `[Authorize]`
  - Returns `FileStreamResult` (image/jpeg etc.)
- [ ] `GET /api/v1/applications/{id}/proof-records/{recordId}/photos/download-all` — `[Authorize]`
  - Returns `FileStreamResult` (application/zip)

### Authorization
- [ ] All authenticated users

---

## API Contract

```
GET /api/v1/applications/{id}/proof-records/{recordId}/photos/{photoId}
```
Returns: file binary with correct Content-Type header.

```
GET /api/v1/applications/{id}/proof-records/{recordId}/photos/download-all
```
Returns: `application/zip`, filename `proof-photos-{recordId}.zip`.

| Response | Description |
|---|---|
| **200** | File stream |
| **404** | Photo not found |
| **401** | Not authenticated |
