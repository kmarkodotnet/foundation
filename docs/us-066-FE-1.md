## Story

> As an **elnök or megtekintő**, I want to view and download proof photos to verify performance.

---

## Scope

Lightbox photo viewer on proof record cards. Individual download button per photo. "Download all as ZIP" button per proof record.

---

## Implementation Checklist

### Feature Module
- [ ] `ProofRecordCardComponent` (part of `StepProofRecordsComponent`):
  - Bélyegképek rács nézet
  - Kattintásra: lightbox megnyílik (ngx-lightbox vagy Angular CDK overlay)
  - Lightbox: bal/jobb nyilak, zoom, letöltés gomb
  - „Összes letöltése ZIP-ben" gomb

### Service
- [ ] `ProofRecordService.getPhoto(appId: string, recordId: string, photoId: string): Observable<Blob>` — GET /api/v1/applications/:id/proof-records/:recordId/photos/:photoId, `{ responseType: 'blob' }`; a Bearer token az `AuthInterceptor`-on keresztül megy; a visszakapott Blob-ból `URL.createObjectURL()` segítségével kell az `<img src>`-t feltölteni — közvetlen URL-string nem használható, mert az `<img>` tag nem viszi az Authorization headert
- [ ] `ProofRecordService.downloadAll(appId: string, recordId: string): Observable<Blob>` — GET /proof-records/:recordId/photos/download-all
- [ ] Photo download: blob letöltés után `URL.createObjectURL(blob)` → `<a download>` trigger

### State / Data Flow
- [ ] Blob letöltés után: `URL.createObjectURL(blob)` → trigger download
- [ ] Lightbox navigáció: előző/következő fotók
