## Story

> As a **pénzügyes**, I want to record the settlement submitted to the granter so that the financial and content report is documented.

---

## Scope

Creates or updates the Settlement record for the application's Settlement workflow step. Sends approval notification to Elnök. Warns when invoice coverage is below 80% of awarded amount.

## API Contract

```
POST /api/v1/applications/{id}/workflow/settlement/request-approval
```
No body. Returns 204. Sends approval notification to all Elnök users.

```
PUT /api/v1/applications/{id}/settlement
```

### Request Body
```json
{
  "settlementDate": "2025-12-15",
  "settlementMethodId": "uuid",
  "description": "Pénzügyi és tartalmi elszámolás benyújtva.",
  "notes": null
}
```

### Response: `SettlementDto`
```json
{
  "id": "uuid",
  "settlementDate": "2025-12-15",
  "settlementMethodId": "uuid",
  "description": "...",
  "invoiceCoveragePercent": 75,
  "hasLowCoverageWarning": true
}
```

| Response | Description |
|---|---|
| **200** | SettlementDto |
| **422** | Application not WON or Settlement step not active |
