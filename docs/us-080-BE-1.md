## Story

> As a **pályázati munkatárs**, I want to upload a document to a workflow step so that all related paperwork is stored in one place.

---

## Scope

Creates a Document record with file storage for any workflow step. Validates file type and size. Returns document metadata with download URL.

## API Contract

```
POST /api/v1/applications/{id}/documents
Content-Type: multipart/form-data
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `workflowStepId` | `Guid?` | No | — |
| `documentTypeId` | `Guid` | Yes | Must exist in CodeList |
| `displayName` | `string?` | No | Max 255 chars |
| `file` | `IFormFile` | Yes | Allowed types; ≤50 MB |

| Response | Description |
|---|---|
| **201** | DocumentDto |
| **422** | Invalid file type or size exceeded |

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `File` | Allowed MIME type | „Ez a fájlformátum nem támogatott. Megengedett formátumok: PDF, DOCX, XLSX, JPG, PNG, TIFF, MSG, EML." |
| `File` | Size ≤ 50 MB | „A fájl mérete meghaladja a 50 MB-os korlátot." |
| `DocumentTypeId` | `NotEmpty()` | „A dokumentum típusa kötelező." |
