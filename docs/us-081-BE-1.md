## Story

> As any **jogosult felhasználó**, I want to download or preview documents so that I can access them from the browser.

---

## Scope

Secure file download endpoint. Files served only through authenticated API, not via direct URL. Content-Disposition header set for browser inline display (PDF) or download.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `DownloadDocumentQuery` — Guid DocumentId → returns file stream, content type, filename
- [ ] Uses `IFileStorageService.ReadAsync(path): Task<Stream>`

### API Layer
- [ ] `GET /api/v1/applications/{id}/documents/{documentId}/download` — `[Authorize]`
  - Returns `FileStreamResult` with correct Content-Type
  - PDF: `Content-Disposition: inline`
  - Others: `Content-Disposition: attachment; filename="..."`

### Authorization
- [ ] All authenticated users

---

## API Contract

```
GET /api/v1/applications/{id}/documents/{documentId}/download
```

| Response | Description |
|---|---|
| **200** | File stream with Content-Type header |
| **404** | Document not found |
| **401** | Not authenticated |
