## Story

> As any **jogosult felhasználó**, I want to download or preview documents from the browser.

---

## Scope

Download and preview actions in `DocumentListComponent`. PDF opens inline; images open in lightbox; other files trigger browser download via blob.

---

## Implementation Checklist

### Feature Module
- [ ] `DocumentListComponent` — dokumentumonként:
  - „Letöltés" gomb → blob letöltés (`URL.createObjectURL`)
  - „Előnézet" gomb (JPG/PNG: lightbox; PDF: iframe/tab megnyitás)
  - Fájltípus ikon a fájlnév előtt

### Service
- [ ] `DocumentService.downloadDocument(appId: string, documentId: string): Observable<Blob>` — GET /api/v1/applications/:id/documents/:documentId/download
- [ ] Blob → `<a>` click trigger a letöltéshez

### State / Data Flow
- [ ] PDF: `window.open(blobUrl)` inline megnyitás
- [ ] Kép: lightbox (megosztott a ProofRecord lightboxszal)
- [ ] Egyéb: `<a download>` trigger

---

## API Consumed

| Method | URL | Body | Response |
|---|---|---|---|
| `GET` | `/api/v1/applications/:id/documents/:documentId/download` | — | Blob |
