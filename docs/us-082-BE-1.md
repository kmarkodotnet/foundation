## Story

> As a **pályázati munkatárs**, I want to upload a new version of an existing document so that the latest version is available while previous versions are preserved.

---

## Scope

Creates a new Document record with Version+1 linked to the parent document. Sets old document IsArchived=true, new one IsArchived=false (Active). Version history endpoint.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Documents/Commands/UploadDocumentVersion/UploadDocumentVersionCommand.cs` — Guid ParentDocumentId, IFormFile File, string? DisplayName
- [ ] Handler:
  - Loads parent Document
  - Sets parent IsArchived=true
  - Creates new Document with Version=parent.Version+1, ParentDocumentId=parent.Id
  - Saves new file via IFileStorageService
- [ ] Query: `GetDocumentVersionsQuery` — Guid DocumentId → returns all versions (current + archived)
- [ ] `DocumentVersionDto`: Id, Version, OriginalFileName, UploadedAt, UploadedByName, IsArchived

### API Layer
- [ ] `POST /api/v1/applications/{id}/documents/{documentId}/versions` (multipart) — `[RequireRole(Admin, PalyazatiMunkatars)]`
  - Returns `201 Created` with new `DocumentDto`
- [ ] `GET /api/v1/applications/{id}/documents/{documentId}/versions` — `[Authorize]`
  - Returns `DocumentVersionDto[]`

### Authorization
- [ ] Upload new version: Admin, PalyazatiMunkatars
- [ ] View versions: all authenticated

---

## API Contract

```
POST /api/v1/applications/{id}/documents/{documentId}/versions
Content-Type: multipart/form-data
```

| Field | Type | Required |
|---|---|---|
| `file` | `IFormFile` | Yes |
| `displayName` | `string?` | No |

| Response | Description |
|---|---|
| **201** | New DocumentDto (active version) |
| **422** | Invalid file type/size |
