## Story

> As an **adminisztrátor**, I want to archive a document so that mistakenly uploaded or outdated files disappear from the active list without being deleted.

---

## Scope

Sets Document.IsArchived=true. File is NOT physically deleted. Archived documents hidden from default list query but accessible via filter.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Documents/Commands/ArchiveDocument/ArchiveDocumentCommand.cs` — Guid ApplicationId, Guid DocumentId; implements `IApplicationCommand`
- [ ] Handler: sets IsArchived=true; file remains on disk
- [ ] `GetDocumentsQuery` (extend): add `includeArchived` filter param (default false)

### API Layer
- [ ] `PATCH /api/v1/applications/{id}/documents/{documentId}/archive`
  - `[RequireRole(UserRole.Admin)]`
  - Returns `204 No Content`

### Authorization
- [ ] Admin only

---

## API Contract

```
PATCH /api/v1/applications/{id}/documents/{documentId}/archive
```

| Response | Description |
|---|---|
| **204** | Archived |
| **403** | Not Admin |
