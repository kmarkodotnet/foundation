## Story

> As an **adminisztrátor**, I want to archive a document so that it disappears from the active list without being deleted.

---

## Scope

Archive action in `DocumentListComponent` visible to Admin only. Archived documents shown with grey styling when "Show archived" filter is active.

---

## Implementation Checklist

### Feature Module
- [ ] `DocumentListComponent`:
  - „Archiválás" ikon (`*hasRole="['Admin']"`)
  - Archiválandó dokumentum szürkítve + „Archivált" badge
  - „Archivált is megjelenítése" szűrő toggle (MatSlideToggle)
  - Archivált dokumentumok alapértelmezetten nem jelennek meg

### Service
- [ ] `DocumentService.archiveDocument(appId: string, documentId: string): Observable<void>` — PATCH /api/v1/applications/:id/documents/:documentId/archive
- [ ] `DocumentService.getDocuments(appId, stepId?, includeArchived?)` — extend with `includeArchived` param

### Authorization
- [ ] „Archiválás": `*hasRole="['Admin']"`

### State / Data Flow
- [ ] Archiválás után: dokumentum eltűnik az aktív listából (ha toggle off)
- [ ] Toast „Dokumentum archiválva."
