## Story

> As a **pályázati munkatárs**, I want to manually record an email related to the grant so that communication history is traceable in the file.

---

## Scope

Creates an EmailRecord linked to an Application and optionally a WorkflowStep. Mandatory fields: subject, sender, sent date, direction. Optional .eml/.msg file attachment handled in US-091.

## API Contract

```
POST /api/v1/applications/{id}/emails
```

### Request Body
```json
{
  "workflowStepId": "uuid",
  "subject": "Pályázat eredményéről értesítés",
  "senderEmail": "palyaztato@example.hu",
  "sentDate": "2025-08-05",
  "direction": "Incoming",
  "contentSummary": "Nyert pályázatról tájékoztatás."
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `subject` | `string` | Yes | NotEmpty |
| `senderEmail` | `string` | Yes | Valid email format |
| `sentDate` | `DateOnly` | Yes | — |
| `direction` | `string` | Yes | Incoming or Outgoing |

| Response | Description |
|---|---|
| **201** | EmailRecordDto |
