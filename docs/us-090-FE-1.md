# us-090-FE-1 | FE: Record Email Manually

**Story:** US-090 | [E-mail] E-mail manuális rögzítése pályázati lépéshez  
**Layer:** Frontend  
**Priority:** Közepes  
**Size estimate:** M  
**Sprint:** Sprint 5  
**Depends on:** us-090-BE-1.md, us-010-FE-1.md  
**Blocks:** us-091-FE-1.md, us-092-FE-1.md

---

## Story

> As a **pályázati munkatárs**, I want to manually record an email so that communication history is in the file.

---

## Scope

Shared `EmailRecordComponent` reused across all workflow step views. Form with required fields. Email list in chronological order (newest first).

---

### Service
- [ ] `EmailService.getEmails(appId: string, workflowStepId?: string): Observable<EmailRecordDto[]>` — GET /api/v1/applications/:id/emails

- [ ] `CreateEmailRequest` TypeScript interfész:
  ```ts
  interface CreateEmailRequest {
    subject: string;
    senderEmail: string;
    sentDate: string; // ISO date
    direction: 'In' | 'Out';
    contentSummary?: string;
    workflowStepId?: string;
  }
  ```
