## Story

> As a **pályázati munkatárs**, I want to upload an .eml or .msg file so that the original email content is preserved.

---

## Scope

Attaches an .eml or .msg file to an existing EmailRecord. Parses .eml for preview (subject, from, date, body). Secure download endpoint.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Emails/Commands/AttachEmailFile/AttachEmailFileCommand.cs` — Guid EmailRecordId, IFormFile File
- [ ] Handler:
  - Validates file type (.eml or .msg only), size ≤ 50 MB
  - Saves file via IFileStorageService
  - Updates EmailRecord.AttachmentFileId
  - For .eml: parses using MimeKit → stores parsed preview data (from, subject, date, text body)
- [ ] Query: `GetEmailPreviewQuery` — Guid EmailRecordId → returns parsed EmlPreviewDto
- [ ] `EmlPreviewDto`: From, Subject, Date, Body (text)

### Infrastructure Layer
- [ ] `IEmailParser.Parse(Stream stream): EmlPreviewDto` (MimeKit implementation)
- [ ] Storage path: `/data/uploads/{year}/{appId}/emails/{uuid}_{filename}`

### API Layer
- [ ] `POST /api/v1/applications/{id}/emails/{emailId}/attachment` (multipart) — `[RequireRole(Admin, PalyazatiMunkatars)]`
  - Returns `200 OK` with `EmailRecordDto`
- [ ] `GET /api/v1/applications/{id}/emails/{emailId}/preview` — `[Authorize]`
  - Returns `EmlPreviewDto`
- [ ] `GET /api/v1/applications/{id}/emails/{emailId}/download` — `[Authorize]`
  - Returns file stream

### Authorization
- [ ] Upload: Admin, PalyazatiMunkatars
- [ ] Preview/Download: all authenticated
