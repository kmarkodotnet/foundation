## Story

> As a **pályázati munkatárs**, I want to delete a mistakenly recorded email record.

---

## Scope

Delete action on email record cards, visible only to the creator and Admin. Confirmation dialog before deletion.

---

## Implementation Checklist

### Feature Module
- [ ] `EmailRecordComponent` — törlés ikon kártyákon:
  - Csak saját rekord esetén látható (currentUser.id == email.createdById) VAGY `*hasRole="['Admin']"`
  - ConfirmDialogComponent: tárgy megjelenítve

### Service
- [ ] `EmailService.deleteEmail(appId: string, emailId: string): Observable<void>` — DELETE /api/v1/applications/:id/emails/:emailId

### State / Data Flow
- [ ] Sikeres törlés: rekord eltávolítása a lokális tömbből (nem API újrahívás)
- [ ] Toast „E-mail rekord törölve."
- [ ] 403 hiba: error toast
