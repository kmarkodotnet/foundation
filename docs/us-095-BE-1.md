## Story

> As any **jogosult felhasználó**, I want to add a comment to a workflow step to record context, observations, and action items.

---

## Scope

Creates a Comment record linked to an Application and optionally a WorkflowStep. Author and timestamp auto-recorded. Returned in chronological order (oldest first).

---

## Implementation Checklist

### Domain Layer
- [ ] `Comment` entity — ApplicationId, WorkflowStepId?, Body (required), IsDeleted, DeletedAt?, AuthorId (FK AppUser), CreatedAt

### Application Layer
- [ ] Command: `Application/Comments/Commands/AddComment/AddCommentCommand.cs` — Guid ApplicationId, Guid? WorkflowStepId, string Body
- [ ] Handler: creates Comment, AuthorId = ICurrentUserService.UserId
- [ ] Validator: Body NotEmpty, MaxLength(2000)
- [ ] Query: `GetCommentsQuery` — by ApplicationId, optional WorkflowStepId, ordered by CreatedAt asc
- [ ] `CommentDto`: Id, Body, AuthorId, AuthorName, AuthorAvatarUrl, CreatedAt, IsDeleted
- [ ] Implements `IApplicationCommand`

### Infrastructure Layer
- [ ] EF Core config: `CommentConfiguration.cs`
- [ ] Migration: `AddCommentTable`

### API Layer
- [ ] Actions in `CommentController`:
  - `GET /api/v1/applications/{id}/comments?stepId=` — `[Authorize]`
  - `POST /api/v1/applications/{id}/comments` — `[Authorize]` (all roles can comment)
  - Returns `201 Created` with `CommentDto`

### Authorization
- [ ] All authenticated users

---

## API Contract

```
POST /api/v1/applications/{id}/comments
```

### Request Body
```json
{
  "workflowStepId": "uuid",
  "body": "A beadási határidőt meg kell erősíteni a pályáztatóval."
}
```

| Field | Type | Required | Validation |
|---|---|---|---|
| `body` | `string` | Yes | NotEmpty, Max 2000 chars |
| `workflowStepId` | `Guid?` | No | — |

| Response | Description |
|---|---|
| **201** | CommentDto |

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
