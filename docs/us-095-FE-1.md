## Story

> As any **jogosult felhasználó**, I want to add comments to a workflow step.

---

## Scope

Shared `CommentSectionComponent` reused across all workflow step views. Chat-like layout. Single-click to show input field.

---

## Implementation Checklist

### Shared Component
- [ ] `src/app/shared/components/comment-section/comment-section.component.ts` + `.html` + `.scss`
  - Chat-szerű elrendezés: avatar, szerző neve, dátum, szöveg
  - „Megjegyzés hozzáadása" link/gomb → textarea megjelenik
  - Textarea (max 2000 karakter, counter)
  - „Küldés" gomb (disabled ha üres)
  - `@Input() appId: string`
  - `@Input() stepId?: string`
  - Mentés után textarea kiürül, megjegyzés azonnal megjelenik

### Service
- [ ] `CommentService.getComments(appId: string, stepId?: string): Observable<CommentDto[]>` — GET /api/v1/applications/:id/comments
- [ ] `CommentService.addComment(appId: string, data: AddCommentRequest): Observable<CommentDto>` — POST /api/v1/applications/:id/comments
- [ ] `AddCommentRequest` TypeScript interfész:
  ```ts
  interface AddCommentRequest {
    body: string;
    workflowStepId?: string; // BE mezőnév: workflowStepId (nem stepId)
  }
  ```

### State / Data Flow
