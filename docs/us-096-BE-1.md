## Story

> As a **megjegyzést rögzítő felhasználó**, I want to edit or delete my own comment to fix typos or withdraw outdated notes.

---

## Scope

Update and soft-delete Comment. Only comment author or Admin can modify/delete. Deleted comments show placeholder text.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Application/Comments/Commands/UpdateComment/UpdateCommentCommand.cs` — Guid ApplicationId, Guid CommentId, string Body; implements `IApplicationCommand`
- [ ] Handler: checks CurrentUserId == Comment.AuthorId OR Admin; updates Body
- [ ] Command: `Application/Comments/Commands/DeleteComment/DeleteCommentCommand.cs` — Guid ApplicationId, Guid CommentId; implements `IApplicationCommand`
- [ ] Handler: same auth check; sets IsDeleted=true, DeletedAt=now, Body="[Megjegyzés törölve]"

### API Layer
- [ ] `PUT /api/v1/applications/{id}/comments/{commentId}` — `[Authorize]`
  - Returns `200 OK` with `CommentDto`
- [ ] `DELETE /api/v1/applications/{id}/comments/{commentId}` — `[Authorize]`
  - Returns `204 No Content`

### Authorization
- [ ] Author or Admin (enforced in handler)

---

## API Contract

```
PUT /api/v1/applications/{id}/comments/{commentId}
```

### Request Body
```json
{ "body": "Javított megjegyzés szövege." }
```

```
DELETE /api/v1/applications/{id}/comments/{commentId}
```

| Response | Description |
|---|---|
| **200** | Updated CommentDto |
| **204** | Deleted |
| **403** | Not author or Admin |
