## Story

> As a **megjegyzést rögzítő felhasználó**, I want to edit or delete my own comment.

---

## Scope

Edit and delete actions on own comment bubbles in `CommentSectionComponent`. Deleted comments show placeholder text in-place.

---

## Implementation Checklist

### Shared Component
- [ ] `CommentSectionComponent` — saját megjegyzéseknél:
  - „Szerkesztés" ikon → inline szerkesztés (textarea az üzeneten)
  - „Törlés" ikon → Confirm dialog
  - Törölt megjegyzés: „[Megjegyzés törölve]" szürke szöveg
  - Más felhasználó megjegyzésein ikonok nem jelennek meg (kivéve Admin)

### Service
- [ ] `CommentService.updateComment(appId: string, commentId: string, body: string): Observable<CommentDto>` — PUT /api/v1/applications/:id/comments/:commentId
- [ ] `CommentService.deleteComment(appId: string, commentId: string): Observable<void>` — DELETE /api/v1/applications/:id/comments/:commentId

### Authorization
- [ ] Szerkesztés/törlés ikon: csak ha `currentUser.id == comment.authorId` VAGY Admin

---

## API Consumed

| Method | URL | Body | Response |
|---|---|---|---|
| `PUT` | `/api/v1/applications/:id/comments/:commentId` | `{ body }` | `CommentDto` |
| `DELETE` | `/api/v1/applications/:id/comments/:commentId` | — | 204 |

## UX / Behaviour Specification

| Scenario | Expected behaviour |
|---|---|
| Szerkesztés ikon kattintás | Inline textarea nyílik előre kitöltve |
| Mentés szerkesztés után | PUT kérés, megjegyzés frissül |
| Törlés ikon | ConfirmDialog |
| Törlés megerősítve | DELETE, „[Megjegyzés törölve]" szöveg megjelenik |
| Más felhasználó megjegyzése | Ikonok nem láthatók |
