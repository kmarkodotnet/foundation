## Story

> As a **pályázati munkatárs**, I want to create a new granter so that reusable, consistent partner data is available for future applications.

---

## Scope

Creates a Granter entity. Name must be unique. Immediately available in application creation autocomplete.

---

## Implementation Checklist

### Domain Layer
- [ ] `Granter` entity — Name (required, unique), Description?, Phone?, Email?, IsActive (bool, default true), IsDeleted

### Application Layer
- [ ] Command: `Granters/Commands/CreateGranter/CreateGranterCommand.cs` — string Name, string? Description, string? Phone, string? Email
- [ ] Handler: checks name uniqueness; creates Granter
- [ ] Validator: Name NotEmpty, unique; Email valid format if provided
- [ ] Query: `GetGrantersQuery` — list (active only by default); supports search term for autocomplete
- [ ] Query: `GetGranterDetailQuery` — Guid GranterId
- [ ] `GranterDto`: Id, Name, Description, Phone, Email, IsActive

### Infrastructure Layer
- [ ] EF Core config: `GranterConfiguration.cs`
- [ ] Migration: `AddGranterTable`
- [ ] Unique index on Name (case-insensitive)

### API Layer
- [ ] `GranterController`:
  - `GET /api/v1/granters?search=&includeInactive=` — `[Authorize]`
  - `GET /api/v1/granters/{id}` — `[Authorize]`
  - `POST /api/v1/granters` — `[RequireRole(Admin, PalyazatiMunkatars)]`
  - Returns `201 Created` with `GranterDto`

### Authorization
- [ ] Read: all authenticated
- [ ] Create: Admin, PalyazatiMunkatars

---

## API Contract

```
POST /api/v1/granters
```

### Request Body
```json
{
  "name": "NKA – Nemzeti Kulturális Alap",
  "description": "Kulturális pályázatok",
  "phone": "+36-1-123-4567",
  "email": "info@nka.hu"
}
```

| Response | Description |
|---|---|
| **201** | GranterDto |
| **422** | Name already exists |

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `Name` | `NotEmpty()` | „A pályáztató neve kötelező." |
| `Name` | Unique | „Ez a pályáztató már szerepel a rendszerben." |
| `Email` | `EmailAddress() when not empty` | „Érvénytelen e-mail cím formátum." |
