## Story

> As an **adminisztrátor**, I want to deactivate a granter so that inactive ones don't clutter selection lists.

---

## Scope

Deactivate/reactivate toggle on the Granter detail page. Warning if linked active applications exist. Inactive granter marked in application views.

---

## Implementation Checklist

### Feature Module
- [ ] `GranterDetailComponent` — „Inaktiválás" gomb (`*hasRole="['Admin']"`):
  - ConfirmDialog: ha van aktív pályázat → figyelmeztetés megjelenik a dialogban
  - Sikeres inaktiválás után: „Aktiválás" gomb jelenik meg, inaktív badge
- [ ] `GrantersListComponent` — inaktív pályáztató szürkítve + „(inaktív)" badge
- [ ] Application autocomplete: `includeInactive=false` alapértelmezetten

### Service
- [ ] `GranterService.deactivateGranter(id: string): Observable<GranterDto>` — PATCH /api/v1/granters/:id/deactivate
- [ ] `GranterService.activateGranter(id: string): Observable<GranterDto>` — PATCH /api/v1/granters/:id/activate
