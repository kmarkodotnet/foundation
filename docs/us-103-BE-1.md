## Story

> As any **jogosult felhasználó**, I want to view a granter's details and all linked applications to review our history with that organization.

---

## Scope

Granter detail query returns full granter data plus a list of linked applications (name, status, awarded amount).

---

## Implementation Checklist

### Application Layer
- [ ] Query: `GetGranterDetailQuery` — Guid GranterId
- [ ] Handler: loads Granter + joined Applications list
- [ ] `GranterDetailDto`:
  - Id, Name, Description, Phone, Email, IsActive
  - `Applications: GranterApplicationSummaryDto[]` (ApplicationId, Title, Status, AwardedAmount)

### API Layer
- [ ] `GET /api/v1/granters/{id}` — `[Authorize]`
  - Returns `GranterDetailDto`

### Authorization
- [ ] All authenticated users

---

## API Contract

```
GET /api/v1/granters/{id}
```

### Response
```json
{
  "id": "uuid",
  "name": "NKA",
  "isActive": true,
  "applications": [
    { "id": "uuid", "title": "Nyári tábor 2025", "status": "WON", "awardedAmount": 1000000 }
  ]
}
```
