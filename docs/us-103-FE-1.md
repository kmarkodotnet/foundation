## Story

> As any **jogosult felhasználó**, I want to view a granter's details and all linked applications.

---

## Scope

Granter detail page at `/granters/:id`. Shows granter info and linked applications table with navigation links.

---

## Implementation Checklist

### Feature Module
- [ ] `GranterDetailComponent` — `/granters/:id` route:
  - Fejléc: Megnevezés, Státusz (aktív/inaktív badge)
  - Adatok kártya: Leírás, Telefonszám, E-mail cím
  - Kapcsolt pályázatok lista (MatTable): Pályázat neve (link), Státusz badge, Elnyert összeg
  - „Szerkesztés" gomb (US-101 scope)
  - „Inaktiválás" gomb (US-102 scope)

### Service
- [ ] `GranterService.getGranterDetail(id: string): Observable<GranterDetailDto>` — GET /api/v1/granters/:id

### Navigation
- [ ] Pályáztató neve kattintható a listanézetből
- [ ] Kapcsolt pályázat sorból navigáció `/applications/:id`-ra
