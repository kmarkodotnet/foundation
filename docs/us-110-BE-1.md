## Story

> As a **pályázati munkatárs or pénzügyes**, I want to register a new vendor so that accurate partner data is available for vendor contracts and invoices.

---

## Scope

Creates a Vendor entity. Name must be unique. Non-blocking warning for invalid tax number format.

---

## Implementation Checklist

### Domain Layer
- [ ] `Vendor` entity — Name (required, unique), TaxNumber?, Address?, Phone?, Email?, IsActive (bool, default true), IsDeleted

### Application Layer
- [ ] Command: `Vendors/Commands/CreateVendor/CreateVendorCommand.cs` — string Name, string? TaxNumber, string? Address, string? Phone, string? Email
- [ ] Handler: checks name uniqueness; creates Vendor
- [ ] Validator: Name NotEmpty, unique; Email valid if provided; TaxNumber format warning (not blocking — return warning in response)
- [ ] Query: `GetVendorsQuery` — list (active only by default), search term for autocomplete
- [ ] `VendorDto`: Id, Name, TaxNumber, Address, Phone, Email, IsActive
- [ ] `CreateVendorResult`: VendorDto + bool HasTaxNumberWarning

### Infrastructure Layer
- [ ] EF Core config: `VendorConfiguration.cs`
- [ ] Migration: `AddVendorTable`
- [ ] Unique index on Name

### API Layer
- [ ] `VendorController`:
  - `GET /api/v1/vendors?search=&includeInactive=` — `[Authorize]`
  - `POST /api/v1/vendors` — `[RequireRole(Admin, PalyazatiMunkatars, Penzugyes)]`
  - Returns `201 Created` with `VendorDto` (+ `taxNumberWarning` header if applicable)

### Authorization
- [ ] Read: all authenticated
- [ ] Write: Admin, PalyazatiMunkatars, Penzugyes

---

## API Contract

```
POST /api/v1/vendors
```

### Request Body
```json
{
  "name": "Acme Kft.",
  "taxNumber": "12345678-1-23",
  "address": "1011 Budapest, Fő u. 1.",
  "phone": "+36-1-123-4567",
  "email": "info@acme.hu"
}
```

| Response | Description |
|---|---|
| **201** | VendorDto |
| **422** | Name already exists |

---

## FluentValidation Rules

| Field | Rule | Error message |
|---|---|---|
| `Name` | `NotEmpty()` | „A cég neve kötelező." |
| `Name` | Unique | „Ez a szerződő cég már szerepel a rendszerben." |
| `Email` | `EmailAddress() when not empty` | „Érvénytelen e-mail cím formátum." |
| `TaxNumber` | Regex `^\d{8}-\d{1}-\d{2}$` (warning only) | „Az adószám formátuma nem megfelelő (pl. 12345678-1-23)." |
