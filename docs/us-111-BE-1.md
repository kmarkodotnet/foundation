## Story

> As any **jogosult felhasználó**, I want to view a vendor's details and all linked contracts to understand our financial relationship with them.

---

## Scope

Vendor detail query returns full vendor data plus linked vendor contracts grouped by application, with a summary panel.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `GetVendorDetailQuery` — Guid VendorId
- [ ] Handler: loads Vendor + VendorContracts with Application names
- [ ] `VendorDetailDto`:
  - Id, Name, TaxNumber, Address, Phone, Email, IsActive
  - `Contracts: VendorContractSummaryDto[]` (ApplicationId, ApplicationTitle, Amount, ContractDate)
  - `Summary: { TotalContracts: int, TotalAmount: decimal }`
- [ ] Command: `Vendors/Commands/UpdateVendor/UpdateVendorCommand.cs` — Guid VendorId, string Name, string? TaxNumber, string? Address, string? Phone, string? Email
- [ ] Handler: loads Vendor, updates fields; validates Name uniqueness excluding self (same guard as CreateVendor)
- [ ] `UpdateVendorCommandValidator` — same rules as CreateVendorCommandValidator

### API Layer
- [ ] `GET /api/v1/vendors/{id}` — `[Authorize]`
  - Returns `VendorDetailDto`
- [ ] `PUT /api/v1/vendors/{id}` — `[RequireRole(Admin, PalyazatiMunkatars)]`
  - Returns `200 OK` with `VendorDetailDto`
  - 422 if name already taken by another vendor

### Authorization
- [ ] All authenticated users

## API Contract

```
GET /api/v1/vendors/{id}
```

### Response
```json
{
  "id": "uuid",
  "name": "Acme Kft.",
  "contracts": [
    { "applicationId": "uuid", "applicationTitle": "Nyári tábor", "amount": 250000, "contractDate": "2025-10-01" }
  ],
  "summary": { "totalContracts": 1, "totalAmount": 250000 }
}
```
