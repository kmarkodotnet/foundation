## Story

> As any **jogosult felhasználó**, I want to view a vendor's details and linked contracts.

---

## Scope

Vendor detail page at `/vendors/:id`. Displays vendor info, linked contracts table by application, and a summary panel.

---

## Implementation Checklist

### Feature Module
- [ ] `VendorDetailComponent` — `/vendors/:id` route:
  - Fejléc: Cégnév, Státusz badge
  - Adatok kártya: Adószám, Cím, Telefonszám, E-mail
  - Összesítő panel: szerződések száma, teljes összeg
  - Kapcsolt szerződések lista (MatTable): Pályázat neve (link), Összeg, Dátum
  - „Szerkesztés" és „Inaktiválás" gomb (US-112 scope)

### Service
- [ ] `VendorService.getVendorDetail(id: string): Observable<VendorDetailDto>` — GET /api/v1/vendors/:id
- [ ] `VendorService.updateVendor(id: string, data: UpdateVendorRequest): Observable<VendorDto>` — PUT /api/v1/vendors/:id
- [ ] `UpdateVendorRequest` TypeScript interfész:
  ```ts
  interface UpdateVendorRequest {
    name: string;
    taxNumber?: string;
    address?: string;
    phone?: string;
    email?: string;
  }
  ```
