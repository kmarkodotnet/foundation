## Story

> As an **adminisztrátor**, I want to deactivate a vendor so that inactive companies don't appear in selection lists.

---

## Scope

Sets Vendor.IsActive=false. Inactive vendors excluded from autocomplete. Existing contracts retain vendor name with "(inaktív)" flag.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `Vendors/Commands/DeactivateVendor/DeactivateVendorCommand.cs` — Guid VendorId
- [ ] Handler: sets IsActive=false; returns warning if active contracts exist
- [ ] Command: `Vendors/Commands/ActivateVendor/ActivateVendorCommand.cs` — Guid VendorId

### API Layer
