# us-120-BE-1 | BE: Manage Codelist Items

**Story:** US-120 | [Kódszótár] Kódszótár elemek kezelése  
**Layer:** Backend  
**Priority:** Magas  
**Size estimate:** M  
**Sprint:** Sprint 2  
**Depends on:** —  
**Blocks:** us-121-BE-1.md

---

## Story

> As an **adminisztrátor**, I want to manage codelist items (add, edit, deactivate, reorder) so that value sets in the application stay current.

---

## Scope

CRUD for CodeListItem entities within existing CodeLists. Drag-and-drop SortOrder updates. Inactive items excluded from selection lists. System codelists cannot be deleted.

---

## Implementation Checklist

### Domain Layer
- [ ] `CodeList` entity — Name (required, unique), IsSystem (bool), IsDeleted
- [ ] `CodeListItem` entity — CodeListId, Code (required, unique within list), Name (required), Description?, SortOrder (int), IsActive (bool, default true), IsDeleted
- [ ] Seed data: system codelists (DocumentType, SubmissionMethod, SettlementMethod, ApplicationType)
  - **Note:** `BudgetItemType` is NOT a CodeList — it is a domain enum (`Event`, `Asset`, `Other`) defined in us-050. Do not seed it as a CodeList.

### Application Layer
- [ ] Command: `CodeLists/Commands/CreateCodeListItem/CreateCodeListItemCommand.cs` — Guid CodeListId, string Code, string Name, string? Description
- [ ] Command: `CodeLists/Commands/UpdateCodeListItem/UpdateCodeListItemCommand.cs` — Guid ItemId, string Code, string Name, string? Description
- [ ] Command: `CodeLists/Commands/DeactivateCodeListItem/DeactivateCodeListItemCommand.cs` — Guid ItemId
- [ ] Command: `CodeLists/Commands/ReorderCodeListItems/ReorderCodeListItemsCommand.cs` — Guid CodeListId, List<Guid> OrderedItemIds
- [ ] Handler (Reorder): updates SortOrder based on list position
- [ ] Query: `GetCodeListsQuery` — returns all CodeLists with item count
- [ ] Query: `GetCodeListItemsQuery` — Guid CodeListId, includeInactive bool
- [ ] `CodeListDto`, `CodeListItemDto`

### Infrastructure Layer
- [ ] EF Core config: `CodeListConfiguration.cs`, `CodeListItemConfiguration.cs`
- [ ] Migration: `AddCodeListTables` + seed data migration

### API Layer
- [ ] `CodeListController`:
  - `GET /api/v1/code-lists` — `[Authorize]`
  - `GET /api/v1/code-lists/{id}/items?includeInactive=` — `[Authorize]`
  - `POST /api/v1/code-lists/{id}/items` — `[RequireRole(Admin)]`
  - `PUT /api/v1/code-lists/{id}/items/{itemId}` — `[RequireRole(Admin)]`
  - `PATCH /api/v1/code-lists/{id}/items/{itemId}/deactivate` — `[RequireRole(Admin)]`
  - `PATCH /api/v1/code-lists/{id}/items/{itemId}/activate` — `[RequireRole(Admin)]`
  - `PUT /api/v1/code-lists/{id}/items/reorder` — `[RequireRole(Admin)]`

### Authorization
- [ ] Read: all authenticated
- [ ] Write: Admin only
