## Story

> As an **adminisztrátor**, I want to create a new custom codelist to support the foundation's unique business categories.

---

## Scope

Creates a new non-system CodeList. Custom codelists can be deleted if no items are linked to records in the system.

---

## Implementation Checklist

### Application Layer
- [ ] Command: `CodeLists/Commands/CreateCodeList/CreateCodeListCommand.cs` — string Name
- [ ] Handler: checks name uniqueness; creates CodeList with IsSystem=false
- [ ] Validator: Name NotEmpty, unique
- [ ] Command: `CodeLists/Commands/DeleteCodeList/DeleteCodeListCommand.cs` — Guid CodeListId
- [ ] Handler:
  - IsSystem=true → 422 blocked
  - Check no CodeListItems are referenced by other entities
  - Soft delete all items + CodeList

### API Layer
- [ ] `POST /api/v1/code-lists` — `[RequireRole(Admin)]`
  - Returns `201 Created` with `CodeListDto`
- [ ] `DELETE /api/v1/code-lists/{id}` — `[RequireRole(Admin)]`
  - Returns `204 No Content`

### Authorization
- [ ] Admin only

---

## API Contract

```
POST /api/v1/code-lists
```
```json
{ "name": "Rendezvény típusa" }
```

| Response | Description |
|---|---|
| **201** | CodeListDto |
| **422** | Name already exists |

```
DELETE /api/v1/code-lists/{id}
```

| Response | Description |
|---|---|
| **204** | Deleted |
| **422** | System codelist or has linked records |
