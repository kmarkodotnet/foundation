## Story

> As an **adminisztrátor**, I want to create a new custom codelist.

---

## Scope

"New Codelist" button on the Codelists admin page. Form for codelist name. Delete button for custom (non-system) codelists.

---

## Implementation Checklist

### Feature Module
- [ ] `CodelistsPageComponent` — extend with:
  - „Új kódszótár" gomb → inline form vagy mat-dialog
  - Kódszótár neve mező (required)
  - Rendszer kódszótáraknál: törlés gomb nem jelenik meg
  - Saját kódszótárnál: törlés gomb → ConfirmDialog

### Service
- [ ] `CodeListService.createCodeList(name: string): Observable<CodeListDto>` — POST /api/v1/code-lists
- [ ] `CodeListService.deleteCodeList(id: string): Observable<void>` — DELETE /api/v1/code-lists/:id

---

## API Consumed

| Method | URL | Body | Response |
|---|---|---|---|
| `POST` | `/api/v1/code-lists` | `{ name }` | `CodeListDto` |
| `DELETE` | `/api/v1/code-lists/:id` | — | 204 |
