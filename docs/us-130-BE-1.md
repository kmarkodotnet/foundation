## Story

> As any **jogosult felhasználó**, I want to filter and sort the application list so that I can quickly find the grant I'm looking for.

---

## Scope

Extends `GetApplicationListQuery` with full filtering/sorting capabilities. Default view: active (non-archived, non-closed) applications.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `Applications/Queries/GetApplicationList/GetApplicationListQuery.cs`:
  - `SearchTerm`: string? (matches title, identifier)
  - `GranterId`: Guid?
  - `ApplicationTypeId`: Guid?
  - `Statuses`: List<ApplicationStatus>?
  - `SubmissionDeadlineFrom`: DateOnly?
  - `SubmissionDeadlineTo`: DateOnly?
  - `AwardedAmountMin`: decimal?
  - `AwardedAmountMax`: decimal?
  - `IncludeArchived`: bool (default false)
  - `SortBy`: enum (SubmissionDeadline | AwardedAmount | LastModified | Status)
  - `SortDirection`: Asc | Desc
  - `PageNumber`, `PageSize`
- [ ] Returns: `PagedResult<ApplicationListItemDto>`
- [ ] `ApplicationListItemDto`: Id, Title, Identifier, GranterName, Status, SubmissionDeadline, SpendingDeadline, AwardedAmount, LastModifiedAt, IsDeadlineWarning (7 days), IsDeadlineCritical (1 day overdue), IsSpendingDeadlineWarning (14 days, WON status only)

### API Layer
- [ ] `GET /api/v1/applications` — `[Authorize]`
  - Query string: all filter params
  - Returns paginated `ApplicationListItemDto[]` with total count

### Authorization
- [ ] All authenticated users

---

## API Contract

```
GET /api/v1/applications?search=&granterId=&statuses=WON,LOST&sortBy=SubmissionDeadline&sortDirection=Desc&page=1&pageSize=20
```

### Response
```json
{
  "items": [...],
  "totalCount": 42,
  "page": 1,
  "pageSize": 20
}
```
