## Story

> As any **jogosult felhasználó**, I want to filter and sort the application list.

---

## Scope

Application list page with a collapsible filter panel, active filter badges, sort controls, and URL-synchronized state.

---

## Implementation Checklist

### Feature Module
- [ ] `ApplicationsListComponent` — `/applications` route:
  - Szűrő panel (összecsukható):
    - Szabad szöveg (MatInput, debounce 300ms)
    - Pályáztató (MatSelect, GranterService.getGranters())
    - Típus (MatSelect, CodeListService)
    - Állapot (MatCheckbox multi-select: DRAFT, IN_PROGRESS, SUBMITTED, WON, LOST, CLOSED_WON, CLOSED_LOST)
    - Beadási határidő tartomány (DateRangePicker)
    - Elnyert összeg sáv (két MatInput)
  - Aktív szűrők badge-ek panel felett (egyenként törölhető)
  - „Összes szűrő törlése" gomb
  - MatTable: cím, pályáztató, állapot badge, határidő (piros/sárga ha közel), összeg
  - MatSort fejlécek
  - MatPaginator

- [ ] URL szinkronizáció: ActivatedRoute + Router query params; szűrők visszaállnak navigáció után

### Service
- [ ] `ApplicationsService.getApplications(filters: ApplicationFilters): Observable<PagedResult<ApplicationListItemDto>>` — GET /api/v1/applications

### State / Data Flow
- [ ] RxJS combineLatest: szűrők + lapozás reaktív stream → API hívás
- [ ] Határidő jelzés: sárga (≤7 nap), piros (lejárt)
