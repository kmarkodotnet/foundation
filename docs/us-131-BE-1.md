## Story

> As any **jogosult felhasználó**, I want to search the whole system from a single search box to find any record quickly.

---

## Scope

Full-text search across Applications (title, identifier), Granters (name), and Vendors (name). Results grouped by entity type. Minimum 3 characters.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `Search/Queries/GlobalSearchQuery.cs` — string SearchTerm (min 3 chars)
- [ ] Handler:
  - Queries Applications: WHERE Title ILIKE or Identifier ILIKE
  - Queries Granters: WHERE Name ILIKE
  - Queries Vendors: WHERE Name ILIKE
  - Returns top 5 per group (configurable)
- [ ] `GlobalSearchResultDto`:
  - `Applications: SearchResultItemDto[]` (Id, Title, Status)
  - `Granters: SearchResultItemDto[]` (Id, Name, IsActive)
  - `Vendors: SearchResultItemDto[]` (Id, Name, IsActive)
- [ ] `SearchResultItemDto`: Id, DisplayName, Status?, EntityType, Url (derived)

### API Layer
- [ ] `GET /api/v1/search?q=` — `[Authorize]`
  - Returns `GlobalSearchResultDto`
  - MinLength validation: q must be ≥ 3 chars

### Authorization
- [ ] All authenticated users

---

## API Contract

```
GET /api/v1/search?q=nyári
```

### Response
```json
{
  "applications": [
    { "id": "uuid", "displayName": "Nyári tábor 2025", "status": "WON" }
  ],
  "granters": [],
  "vendors": []
}
```

| Response | Description |
|---|---|
| **200** | GlobalSearchResultDto |
| **422** | Search term < 3 characters |
