## Story

> As any **jogosult felhasználó**, I want to search across the whole system from the navbar.

---

## Scope

Persistent search input in the navbar. Results dropdown grouped by entity type. Navigates to the record on selection.

---

## Implementation Checklist

### Shared Component
- [ ] `src/app/shared/components/global-search/global-search.component.ts` + `.html` + `.scss`
  - Keresőmező a navigációs sávban (ikon + expandable input)
  - Eredmény dropdown (MatAutocomplete vagy custom overlay):
    - Csoportok: Pályázatok / Pályáztatók / Szerződő cégek
    - Minden találat: megnevezés + státusz badge
  - Minimum 3 karakter, debounce 300ms
  - „Nincs találat" üzenet
  - Enter vagy találat kattintás → navigáció

### Service

### State / Data Flow

- [ ] Navigáció: `/applications/:id`, `/granters/:id`, `/vendors/:id`

## UX / Behaviour Specification

| Scenario | Expected behaviour |
|---|---|
| < 3 karakter | Dropdown nem jelenik meg |
| Találat kattintás | Navigáció + dropdown bezárás |
| Nincs találat | „Nem található rekord a(z) „X" kifejezésre." |
| Enter | Navigáció az első találatra |
