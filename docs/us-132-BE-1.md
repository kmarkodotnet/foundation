## Story

> As an **elnök or pénzügyes**, I want to export the filtered application list to Excel for offline analysis.

---

## Scope

Generates an .xlsx file with the same filters as the current list view. Uses ClosedXML or EPPlus. Filename includes date.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `Applications/Queries/ExportApplications/ExportApplicationsQuery.cs` — same filter params as GetApplicationListQuery
- [ ] Handler:
  - Fetches all matching records (no pagination)
  - Generates .xlsx via ClosedXML
  - Columns: ID, Cím, Pályáztató, Típus, Állapot, Beadási határidő, Elnyert összeg, Elköltési határidő, Létrehozva, Módosítva
- [ ] Returns: `ExportResult { byte[] Content, string FileName }`

### Infrastructure Layer
- [ ] NuGet: `ClosedXML` (or `EPPlus`)

### API Layer
- [ ] `GET /api/v1/applications/export?{same filters}` — `[RequireRole(Admin, Elnok, Penzugyes)]`
  - Returns `FileResult` with `Content-Disposition: attachment; filename="palyazatok_YYYYMMDD.xlsx"`
  - Content-Type: `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`

### Authorization
- [ ] Admin, Elnok, Penzugyes

---

## API Contract

```
GET /api/v1/applications/export?statuses=WON
```
Returns: Excel file blob.

| Response | Description |
|---|---|
| **200** | Excel file |
| **403** | Insufficient role |
