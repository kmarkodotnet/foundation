## Story

> As an **elnök or pénzügyes**, I want to export the filtered application list to Excel.

---

## Scope

Export button in the application list toolbar. Passes current filters to the export endpoint. Triggers browser download.

---

## Implementation Checklist

### Feature Module
- [ ] `ApplicationsListComponent` — „Exportálás (.xlsx)" gomb:
  - `*hasRole="['Admin', 'Elnok', 'Penzugyes']"`
  - Disabled ha a lista üres (totalCount === 0)
  - Töltés indikátor exportálás közben
  - Sikeres letöltés: blob → `<a download>` trigger

### Service
- [ ] `ApplicationsService.exportApplications(filters: ApplicationFilters): Observable<Blob>` — GET /api/v1/applications/export (same query params as list)
  - 403 error handler: toast „Exportáláshoz megfelelő jogosultság szükséges." (Admin/Elnök/Pénzügyes szerepkör kell)

### State / Data Flow
- [ ] Blob letöltés: `URL.createObjectURL(blob)` → `<a>` click trigger
- [ ] Filename: `palyazatok_{YYYYMMDD}.xlsx` (from Content-Disposition header vagy kliens-oldalon generált)
