## Story

> As a **pályázati munkatárs**, I want to see deadline warnings in the application list.

---

## Scope

Visual deadline warning in the application list (yellow/red icons). Notifications received via the notification system (US-142 handles the notification panel).

---

## Implementation Checklist

### Feature Module
- [ ] `ApplicationsListComponent` — határidő oszlop:
  - Sárga figyelmeztető ikon: ≤ 7 nap (isDeadlineWarning=true a API response-ban)
  - Piros ikon: 1 nappal lejárt (isDeadlineCritical=true)
  - MatTooltip: „X nap van hátra a beadási határidőig" / „A beadási határidő X napja lejárt"

- [ ] Határidő badge a pályázat részletes nézetén is megjelenik

### State / Data Flow
- [ ] `ApplicationListItemDto.isDeadlineWarning` és `isDeadlineCritical` flag-ek az API response-ban → reaktívan jelenítve meg

---

## UX / Behaviour Specification

| Scenario | Expected behaviour |
|---|---|
| ≤ 7 nap | Sárga figyelmeztető ikon + tooltip |
| Lejárt határidő | Piros ikon + tooltip |
| Normál határidő | Nincs ikon |
