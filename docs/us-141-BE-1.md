## Story

> As a **pénzügyes**, I want to be notified when the spending deadline is approaching so that we ensure invoices are collected and payments made on time.

---

## Scope

Extends the deadline alert Hangfire job to also check SpendingDeadline. Notifies Penzugyes and Elnök 14 days before. Includes financial balance summary in notification body.

---

## Implementation Checklist

### Application Layer
- [ ] `DeadlineAlertJob` (extend from US-140):
  - Also queries: applications WHERE SpendingDeadline = today + 14 days AND Status = WON
  - Creates notifications for Penzugyes + Elnök users
  - Notification body includes: AwardedAmount, TotalInvoiced, Balance
- [ ] `NotificationType.DeadlineSpending` enum value

---

## API Contract
No new endpoints. Handled by the same Hangfire daily job.
