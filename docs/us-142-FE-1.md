## Story

> As any **bejelentkezett felhasználó**, I want to view all my notifications and mark them as read.

---

## Scope

Bell icon with unread badge in navbar. Notifications dropdown/panel. Real-time updates via SignalR. Mark as read on click.

---

## Implementation Checklist

### Shared Component
- [ ] `src/app/shared/components/notification-bell/notification-bell.component.ts` + `.html` + `.scss`
  - Harang ikon a navigációs sávban
  - Olvasatlan count badge (piros, eltűnik ha 0)
  - Kattintásra panel megnyílik (MatMenu vagy overlay)

- [ ] `NotificationsPanelComponent`:
  - Értesítések lista (legújabb felül)
  - Minden értesítésnél: típus ikon, szöveg, időbélyeg, olvasatlan stílus
  - „Összes olvasottnak jelölése" gomb (ha van olvasatlan)
  - Üres állapot: „Nincs új értesítésed"
  - Értesítésre kattintás: olvasottnak jelöl + navigál

### Service
- [ ] `NotificationService.getNotifications(unreadOnly?: boolean): Observable<NotificationDto[]>` — GET /api/v1/notifications
- [ ] `NotificationService.markRead(id: string): Observable<void>` — PATCH /api/v1/notifications/:id/read
- [ ] `NotificationService.markAllRead(): Observable<void>` — PATCH /api/v1/notifications/read-all
- [ ] `NotificationService.connect(): void` — SignalR HubConnection setup
  - On new notification: prepend to list, increment badge

### SignalR
- [ ] `@microsoft/signalr` npm package
- [ ] Hub URL: `/hubs/notifications`
- [ ] JWT token in connection options (withUrl(..., { accessTokenFactory }))
- [ ] Event: `ReceiveNotification` → update local notification list

### State / Data Flow
- [ ] BehaviorSubject: unreadCount$, notifications$
- [ ] ngOnInit: connect SignalR, load initial notifications
