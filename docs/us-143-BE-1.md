## Story

> As an **elnök**, I want to receive a notification when a workflow step requires my approval so that I don't have to keep checking manually.

---

## Scope

Centralizes approval request notification sending. Triggered by RequestApproval commands for Submission, BudgetPlan, and Settlement steps. Creates Notification records + sends email to all Elnök-role users.

---

## Implementation Checklist

### Application Layer
- [ ] `INotificationService.SendApprovalRequestAsync(Guid applicationId, string applicationTitle, string stepName)`:
  - Finds all AppUsers with role=Elnok (and Admin optionally)
  - Creates Notification records for each with Type=ApprovalRequest
  - Sends email via IEmailSender (if ApprovalRequestEmail preference=true)
- [ ] Called from:
  - `RequestApprovalCommandHandler` (Submission step)
  - `RequestApprovalCommandHandler` (BudgetPlan step)
  - `RecordSettlementCommandHandler` (Settlement step)
- [ ] `NotificationType.ApprovalRequest` enum value

### Infrastructure Layer
- [ ] Email template: „Jóváhagyás szükséges: [PályázatCím] – [LépésNév]"
- [ ] Direct link in email body to the workflow step
