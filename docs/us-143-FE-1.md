## Story

> As an **elnök**, I want to receive approval request notifications so that I'm guided directly to where I need to act.

---

## Scope

Approval request notifications appear in the notification panel (US-142). Clicking navigates directly to the relevant workflow step. No additional FE implementation beyond US-142.

---

## Implementation Checklist

### Feature Module
- [ ] `NotificationsPanelComponent` (US-142) — ApprovalRequest típusú értesítések:
  - Speciális ikon (pl. check_circle MatIcon)
  - Kattintásra navigáció a lépésre: `/applications/:entityId?step=submission` stb.
  - Értesítés szövege: „Jóváhagyás szükséges: [PályázatCím] – [LépésNév]"

### State / Data Flow
- [ ] `entityId` → Application ID
- [ ] `entityType=WorkflowStep` → step type meghatározása a notification body-ból
- [ ] Navigáció: `router.navigate(['/applications', entityId], { queryParams: { step: stepType } })`
