## Story

> As an **adminisztrátor**, I want to view all data change events in the audit log to trace who changed what and when.

---

## Scope

AuditLog entity populated by AuditBehaviour in MediatR pipeline. Queryable with filters. Exportable to CSV. Records are immutable.

---

---

## API Contract

```
GET /api/v1/audit-logs?userId=&dateFrom=&entityType=Application&actionType=StatusChange&page=1&pageSize=50
```

### Response
```json
{
  "items": [
    {
      "id": "uuid",
      "timestamp": "2025-10-01T14:30:00Z",
      "userEmail": "user@example.hu",
      "actionType": "StatusChange",
      "entityType": "Application",
      "entityId": "uuid",
      "fieldName": "Status",
      "oldValue": "SUBMITTED",
      "newValue": "WON"
    }
  ],
  "totalCount": 150
}
```
