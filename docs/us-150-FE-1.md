## Story

## Scope

Audit log admin page with filter panel, paginated table, and CSV export.

## Implementation Checklist

### Feature Module
- [ ] `src/app/features/admin/audit-log/audit-log.module.ts` (lazy-loaded, Admin only)
- [ ] `AuditLogPageComponent` — `/admin/audit-log` route:
  - Szűrő panel: felhasználó (MatSelect), dátumtartomány (DateRangePicker), entitás típus, művelettípus
  - MatTable: időbélyeg, felhasználó (e-mail), művelettípus badge, entitás típus, entitás neve/ID, mező, régi → új érték
  - MatPaginator
  - „Exportálás CSV" gomb → `AuditLogService.exportCsv()`

### Service
- [ ] `AuditLogService.getAuditLogs(filters: AuditLogFilters): Observable<PagedResult<AuditLogDto>>` — GET /api/v1/audit-logs
- [ ] `AuditLogService.exportCsv(filters: AuditLogFilters): Observable<Blob>` — GET /api/v1/audit-logs/export
- [ ] `AuditLogDto` TypeScript interfész:
  ```ts
  interface AuditLogDto {

    timestamp: string; // ISO datetime
    userEmail: string;
    actionType: string;
    entityType: string;

    fieldName?: string;
    oldValue?: string;
    newValue?: string;
  }
  ```

  ```ts

  ```
