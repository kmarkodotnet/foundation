## Story

> As an **elnök**, I want to view the modification history of a specific application so that I can understand what changed during its lifecycle.

---

## Scope

Extends the audit log query to filter by ApplicationId. Used for the application detail page "History" tab.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `GetApplicationAuditLogsQuery` — Guid ApplicationId, PageNumber, PageSize
- [ ] Handler: filters AuditLog WHERE EntityId == ApplicationId OR related entities (WorkflowStep, Invoice, etc.) linked to this Application
- [ ] Ordered by Timestamp desc

### API Layer
- [ ] `GET /api/v1/applications/{id}/audit-logs` — `[RequireRole(Admin, Elnok)]`
  - Returns paginated `AuditLogDto[]`

### Authorization
- [ ] Admin, Elnok

---

## API Contract

```
GET /api/v1/applications/{id}/audit-logs?page=1&pageSize=20
```
Returns same `AuditLogDto` structure as global audit log.
