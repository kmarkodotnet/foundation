## Story

> As an **elnök**, I want to see the modification history of a specific application.

---

## Scope

"History" tab on the application detail page, visible to Admin and Elnök. Timeline-style list with expandable entries.

---

## Implementation Checklist

### Feature Module
- [ ] `ApplicationDetailComponent` — „Előzmények" tab (`*hasRole="['Admin', 'Elnok']"`):
  - Timeline lista: időbélyeg, felhasználó, művelettípus, entitás neve
  - Kibontható sorok (MatExpansionPanel): régi → új érték összehasonlítás
  - MatPaginator

### Service
- [ ] `ApplicationsService.getAuditLogs(appId: string, page?: number): Observable<PagedResult<AuditLogDto>>` — GET /api/v1/applications/:id/audit-logs
  - 403 error handler: az „Előzmények" tab elrejthető, de ha programmatikusan hívódna: toast „A pályázat előzményeihez nincs jogosultsága."
