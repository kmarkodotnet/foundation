## Story

> As an **adminisztrátor**, I want to see all registered users with their roles, status, and last login time.

---

## Scope

Returns paginated list of AppUsers. Supports filtering by name, email, role. Visible to Admin only.

---

## Implementation Checklist

### Application Layer
- [ ] Query: `Users/Queries/GetUsersQuery.cs` — SearchTerm?, Role (UserRole?), PageNumber, PageSize
- [ ] Handler: returns paginated AppUser list
- [ ] `UserListItemDto`: Id, FullName, Email, AvatarUrl, Role, IsActive, LastLoginAt

### API Layer
- [ ] `UserController`:
  - `GET /api/v1/users?search=&role=&page=&pageSize=` — `[RequireRole(Admin)]`
  - Returns `PagedResult<UserListItemDto>`

### Authorization
- [ ] Admin only

---

## API Contract

```
GET /api/v1/users?search=kiss&role=PalyazatiMunkatars
```

### Response
```json
{
  "items": [
    {
      "id": "uuid",
      "fullName": "Kiss János",
      "email": "kiss.janos@example.hu",
      "role": "PalyazatiMunkatars",
      "isActive": true,
      "lastLoginAt": "2025-10-01T09:30:00Z"
    }
  ],
  "totalCount": 5
}
```
