## Story

> As an **adminisztrátor**, I want to assign roles to users.

---

## Scope

Inline role selector on each user row in the Users list. Confirmation for role changes. Error handling for last-Admin guard.

---

- [ ] `UsersPageComponent` — soronként „Szerepkör módosítása":

  - 422 hiba: error toast „Nem távolítható el az utolsó Admin szerepköre."
