# us-162-FE-1 | FE: Activate/Deactivate User

**Story:** US-162 | [Admin] Felhasználó inaktiválása és reaktiválása  
**Layer:** Frontend  
**Priority:** Közepes  
**Size estimate:** S  
**Sprint:** Sprint 2  
**Depends on:** us-162-BE-1.md, us-160-FE-1.md  
**Blocks:** —

---

## Story

> As an **adminisztrátor**, I want to deactivate or reactivate users.

---

## Scope

Deactivate/reactivate button on each user row. Self-deactivation attempt shows error. Inactive users visually greyed out.

---

## Implementation Checklist

### Feature Module
- [ ] `UsersPageComponent` — soronként:
  - „Inaktiválás" gomb (ha isActive=true, `*hasRole="['Admin']"`, hidden for currentUser)
  - „Reaktiválás" gomb (ha isActive=false)
  - ConfirmDialog: „Biztosan inaktiválod [Felhasználó neve]?"
  - 422 hiba (saját fiók): error toast „Saját fiókod nem inaktiválhatod."

### Service
- [ ] `UserService.deactivateUser(userId: string): Observable<UserListItemDto>` — PATCH /api/v1/users/:id/deactivate
- [ ] `UserService.activateUser(userId: string): Observable<UserListItemDto>` — PATCH /api/v1/users/:id/activate

### State / Data Flow
- [ ] Sikeres inaktiválás: felhasználó szürkítve jelenik meg
- [ ] Toast „Felhasználó inaktiválva." / „Felhasználó reaktiválva."

---
