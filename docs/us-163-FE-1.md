## Story

> As an **adminisztrátor**, I want to configure system settings so that the system fits the foundation's needs.

---

## Scope

System settings admin page with a simple form. Settings are loaded on page open and saved on submit.

---

## Implementation Checklist

### Feature Module
- [ ] `src/app/features/admin/settings/settings.module.ts` (lazy-loaded, Admin only)
- [ ] `SystemSettingsPageComponent` — `/admin/settings` route:
  - Értesítési előfigyelmeztetés napjai (MatInput number, required, min 1)
  - Elköltési határidő előfigyelmeztetés napjai (MatInput number, required, min 1)

  - Alapértelmezett szerepkör (MatSelect)

### Service
